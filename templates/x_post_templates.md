# X投稿文パターン見本集

CLINEがX投稿文を生成する際に参照するフォーマット集です。
1記事につき1本生成します。

---

## 投稿ルール

- **形式**：Xプレミアムの長文投稿機能を活用し、記事本文に近い読み応えのあるコンテンツとして投稿
- **文字数**：1,500〜3,000文字程度（記事の内容量に応じて調整）
- **CTA末尾にUTM付きカテゴリリンクを必ず含める**（ブランド記事→カテゴリページ、テーマ記事→トップページ）
  - UTMパラメータ: `utm_source=x&utm_medium=social&utm_campaign=magazine`
  - 例: `https://firekids.jp/products/list?category_id=8&utm_source=x&utm_medium=social&utm_campaign=magazine`
  - テーマ/複数ブランド記事: `https://firekids.jp/?utm_source=x&utm_medium=social&utm_campaign=magazine`
- **個別商品リンクは使用しない**（カテゴリページのみ）
- **相場価格は記載しない**
- **ハッシュタグは使わない**
- **トーン**：記事本文と同様の専門性ある文体。ただしX向けに段落を短くし、視認性を高める
- **情報ソース**：記事本文（= FK記事作成用.xlsx の Body 由来）に基づく内容のみ

### レイアウトのポイント

- **段落は短く区切る**：1段落3〜5文。長い段落はX上で読みにくいため分割する
- **セクション区切り線**：「━━━━━━━━━━━━━━」で大きな構成を区切る
- **見出しに絵文字を使用**：各セクションの冒頭に絵文字を置き、スクロール時の視認性を高める
- **空行を多用**：行間を空けてX上での可読性を確保する
- **比較テーブルは箇条書きに変換**：X上ではテーブル表示ができないため、箇条書き形式に変換する

## NG例

### NG1：価格に言及
```
NG: 現在○○万円で取引されています
→ 相場価格は変動するため記載NG
```

### NG2：個別商品への誘導
```
NG: FIRE KIDSで販売中のRef.○○はこちら → [URL]
→ 個別商品リンクはNG
```

### NG3：ハッシュタグを使う
```
NG: #ヴィンテージ時計 #FIREKIDS #オメガ
→ ハッシュタグはリーチを下げるため使用しない
```

### NG4：記事HTMLをそのまま貼り付け
```
NG: Gutenbergブロックタグやメタ情報を含んだままの貼り付け
→ プレーンテキストに変換し、X向けのレイアウトに整形すること
```

### NG5：FK番号を含む
```
NG: FK000123の商品は〜
→ FK番号は一切含めない
```

## 記事からX投稿への変換ガイドライン

1. **記事のH2見出しを長文投稿のセクションに対応させる**（全セクションを含める必要はなく、主要な3〜5セクションを選定）
2. **テーブルは箇条書き（・）に変換する**
3. **HTMLタグ・Gutenbergブロックタグはすべて除去する**
4. **記事の導入文をそのまま活用可能**（X向けに1〜2文に短縮するのが望ましい）
5. **商品データベースからの引用（「」付き記述）は記事と同様に使用可能**
6. **画像は含めない**（テキストのみの投稿とする）
7. **セクション間は空行で区切り、大区分には区切り線（━━━）を使用する**
