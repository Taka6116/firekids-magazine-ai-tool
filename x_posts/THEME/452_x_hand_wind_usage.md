手巻き時計の正しい巻き方｜ヴィンテージを長く使うためのリューズ操作ガイド

ヴィンテージの手巻き時計を手に入れて、「これで合ってるのかな？」とリューズを回しながら不安になった経験はないでしょうか。毎日の巻き上げは、手巻き時計との付き合い方そのものです。強すぎず、弱すぎず、ちょうどよい力加減で回す。そのコツを押さえれば、古い時計も長く気持ちよく使い続けられます。

なぜ「正しい巻き方」が大事なのか

誤った巻き方はゼンマイやリューズまわりを傷めやすく、結果的にオーバーホール費用や部品交換の手間を増やす原因になります。リューズを無造作に握って回したり、斜めに引っぱったりすると、巻き真や内部の歯車に余計な負荷がかかります。

ヴィンテージになるほど部品は細く、経年で金属疲労も進んでいるため、現行モデル以上に扱いに気を配りたいところです。

基本の3ステップ

ステップ1：時計を腕から外して指の腹で挟む
腕に着けたまま巻くとリューズに対して斜めに力がかかりやすく、巻き真に横方向の負荷を与えてしまいます。親指と人差し指の腹でリューズをやさしく挟み、爪で引っかけずに指紋の部分で軽く押さえます。

ステップ2：時計回りに一定のスピードで回す
手巻き時計の巻き上げ方向は、文字盤を見た状態で時計回り（右回り）が基本。スピードはゆっくり・一定が理想です。「ツー、ツー、ツー」と同じリズムで、1回転ずつ区切るように回してみてください。

ステップ3：抵抗を感じたら即ストップ
巻き上げていくと「重くなった」「これ以上巻けない」という感触が指先に伝わってきます。これが巻き止まりで、ここで止めるのが鉄則。無理に押し込むようにさらに巻こうとしてはいけません。

巻き上げの目安と頻度

手巻き時計のパワーリザーブは、おおむね1日半〜2日前後が一つの目安。毎朝決まった時間に巻く習慣にすると、巻き忘れによる停止を避けやすくなります。朝に巻いたほうが「ゼンマイがもっとも元気な状態で日中を過ごせる」ため、精度の安定につながりやすいといえます。

巻き上げ回数は個体によって異なるため、回数を数えるより指先の感触を優先してください。

リューズの段階操作

通常位置（押し込んだ状態）：巻き上げ
リューズを一番奥まで押し込んだ位置がゼンマイ巻き上げ位置。この状態で針が動いてしまう場合は何らかの異常のサインです。

1段引いた位置：時刻合わせまたは日付合わせ
モデルによって割り当てが異なります。デイト付きモデルでは1段目が日付送り（クイックチェンジ）、ノンデイトモデルでは1段引けば時刻合わせ。

2段引いた位置：時刻合わせ
デイト付きモデルでは2段目で時刻合わせ。ヴィンテージ手巻きではハック機能なしの個体もあれば、秒針が止まるハック付きもあります。

操作後はリューズを必ずもとの位置まで押し込むこと。スクリューリューズ式の場合は押し込みながら時計回りに軽く締めるタイプが一般的です。

━━━━━━━━━━━━━━

日付送りの禁止時間帯

多くのヴィンテージ日付付きモデルでは、午後9時ごろから翌午前3時ごろにかけて、内部で日付を1日送る機構（日送り車）が噛み合いはじめています。この時間帯にリューズをクイック日付送り位置まで引いて強制的に日付を変えると、日送り車の歯欠けにつながることがあります。

・時計の針を午前6時〜午後8時の範囲に動かしてから、日付をクイック送りで合わせる
・もしくは、時刻合わせ位置でゆっくり針を24時間回して日付を送る

ヴィンテージ特有の注意点

ゼンマイと香箱の経年劣化
数十年前に作られたゼンマイは、使用頻度や保管状態によって金属疲労が進んでいることがあります。「最近、巻き止まりの感触が浅い」「昨日より持ちが短い」といった変化はサインの可能性。

リューズ・巻き真・パッキンの状態
長年の使用でリューズが削れる、巻き真のネジ山がすり減る、パッキンが硬化するといった症状もよく見られます。スクリューリューズ式は、必ず押し込みながら抵抗のないところから時計回りに軽く締めてください。

代表的な手巻きキャリバーの例
・ロレックス Cal.1225（オイスターデイトRef.6694の搭載機）
・オメガ 30mm系・Cal.284/285系
・セイコー Cal.4420系（44GS）、Cal.5740C（ロードマーベル36000）
・チューダー Ref.7934のような手巻きオイスター系

朝のルーティン例

・時計を腕から外して手に持つ → 巻き真に斜めの力がかかるのを避ける
・指の腹でリューズをつまむ → 爪で引っかけない
・時計回りにゆっくり巻く → 一定のリズムで力みすぎない
・抵抗を感じたら止める → 巻き止まりを超えて回さない
・必要なら時刻・日付を合わせる → 日付操作は昼間の時間帯で
・リューズを通常位置に押し戻す → スクリュー式は軽く締める
・腕に着ける

夜は外して、文字盤を上にして静置するのがおすすめです。

こんな方におすすめしたい

・ヴィンテージ手巻き時計を初めて手にする方 → 基本の3ステップだけ押さえる
・手巻き時計を久しぶりに再稼働させたい方 → まずリューズを巻き止まりまで巻き、1〜2時間様子を見る
・一本を長く、大切に使いたい方 → 毎日の所作ひとつで寿命が変わる
・家族から譲り受けたヴィンテージを持っている方 → 巻き切ろうとしない、日付送りの禁止時間帯を避ける、違和感を感じたら時計師に相談

まとめ

手巻きヴィンテージ時計との付き合いで大事なのは、派手な知識よりも毎日の小さな習慣です。

・腕から外して指の腹でリューズをつまむ
・時計回りに一定のスピードでゆっくり巻く
・抵抗を感じたらそこでやめる
・日付操作は昼間の時間帯に行う
・いつもと違う感触があったら無理せず時計師に相談する

リューズを巻く数秒間は、時計との短い会話のような時間でもあります。その感触を楽しみながら、長く付き合っていただけたら嬉しいです。
