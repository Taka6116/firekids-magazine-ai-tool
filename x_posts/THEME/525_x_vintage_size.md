ヴィンテージ時計のサイズ、手首径から見る選び方

「ヴィンテージって、どれもサイズが小さくて自分の手首だと頼りなく見えるのでは？」——そう感じている方は意外と多いのではないでしょうか。

たしかに現行モデルは40〜43mmが主流で、ヴィンテージの34〜36mmという数字を見ると小さく感じるかもしれません。ただ、実際に手首に合わせてみると、現行と思っていたサイズが大きすぎて窮屈に感じるケースもあります。

ヴィンテージのケース径が小さめな理由

ヴィンテージ時計のケース径が34〜36mm前後に集中しているのは、当時の価値観と装着シーンに理由があります。

スーツの袖口にすっと収まるサイズ感が紳士の腕時計の標準とされていた時代背景があり、装着感の良さと視認性のバランスが取れる範囲として34〜36mm前後が定着しました。

商品データの実測平均でも、ロレックスのヴィンテージは平均34mm前後、オメガで35〜36mm、ロンジンで34mm前後、カルティエで32mm前後、セイコーで37mm前後（5アクタスや150mダイバーなど大型を含む）という分布になっています。

現代の40mm以上が標準化したのは2000年代以降の流れで、それ以前のヴィンテージは「腕に乗るサイズ」「袖口に収まるサイズ」が基本でした。

手首周りからケース径を逆算する

短く言えば、手首周りの細さに対してケース径の比率が大きすぎると窮屈に、小さすぎると物足りなく見えます。

一般的な目安として、

・手首14〜15cm → 31〜34mm
・手首15〜16cm → 33〜36mm
・手首16〜17cm → 35〜38mm
・手首17〜18cm → 37〜40mm
・手首18cm以上 → 38〜42mm

これはあくまで目安で、ラグの長さやラグ幅、文字盤の余白の取り方によっても見え方が変わります。手首から時計がはみ出していないか、ラグの先端が手首の側面に当たっていないかを実際に確認するのが確実です。

ブランド別のサイズ感

ロレックスのヴィンテージは34〜40mmで分布。デイトジャスト Ref.1601は36mm、サブマリーナ Ref.5513は40mm、エアキング Ref.5500／14000は34mm、エクスプローラーI Ref.14270は36mm。用途別にサイズが分かれます。

オメガはコンステレーション C／12角が33〜35mm、シーマスターが34〜36mm、スピードマスター プロフェッショナルが42mm。ドレス系は33〜35mmに集中します。

セイコー／シチズンは平均36〜37mm前後で海外より1〜2mm大きめの傾向。グランドセイコー44GSは36mm、150mダイバーは44mm。

ロンジン／IWC／チューダーはコンクエスト／フラッグシップが33〜35mm、IWC オートマチックが34〜36mm、チューダー オイスタープリンスが34mm。

カルティエはマストタンクLMが縦約30mm×横約23mm、ラグ幅17mm。角型は数字より大きく見える点が選ぶときのポイントです。

ラグ幅と着け心地

ケース径と並んで、ラグ幅も装着感を大きく左右します。

・ラグ幅16〜17mm：ドレスウォッチ（ロンジン フラッグシップ、カルティエ マストタンクLM）
・ラグ幅18〜19mm：3針モデル中心（ロレックス オイスター系、エアキング、IWC オートマチック）
・ラグ幅20mm：定番スポーツ／クロノグラフ（デイトジャスト、サブマリーナ、エクスプローラー）
・ラグ幅22mm：大型ダイバー／クロノグラフ（セイコー 150mダイバー、ホイヤー オータヴィア）

ラグ幅が大きいほどベルトの存在感が増し、手首が太く見える効果があります。

ヴィンテージ時計のサイズは、現行モデルとは異なる基準で設計されています。

34〜36mm前後の数字を「小さい」と感じるか「ちょうどいい」と感じるかは、手首周りやラグ幅、用途次第で変わってきます。数字だけで判断せず、ラグ・トゥ・ラグやラグ幅、装着シーンを総合して見ていくのが、ヴィンテージ時計のサイズ選びの基本です。

袖口にすっと収まる感覚を味わうと、現代の大型ケースとは別の「腕に乗る心地よさ」を発見できるかもしれません。
