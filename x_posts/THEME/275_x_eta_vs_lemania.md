ETA vs レマニア vs バルジュー ヴィンテージクロノグラフCal比較

ヴィンテージクロノグラフの搭載キャリバーを深く読むと、多くのモデルが「ETA」「レマニア」「バルジュー」の3つのメーカーに行き着きます。これらはいずれも独立系のムーブメント供給メーカー（エブーシュメーカー）で、ロレックスやオメガ、ホイヤーといった著名ブランドの多くが自社ケースにこれらのキャリバーを搭載してきました。

3社のキャリバーの特徴と代表的な搭載モデルを比較します。

━━━━━━━━━━━━━━

レマニア系キャリバー ― コラムホイール式の代名詞

レマニアはスイスのル・サンティエを拠点とする高級クロノグラフキャリバーの供給メーカーです。その代表格がオメガ Cal.321で、手巻き・17石・18,000振動/時という仕様で1946年に導入されました。

Cal.321の最大の特徴はコラムホイール式のクロノグラフ制御機構を採用している点です。コラムの山と谷でレバー類の位置を切り替えるこの機構は、プッシャーの操作感が滑らかで、製造には高い加工精度が求められます。1957年のスピードマスター初代CK2915から1968年まで搭載され続けました。

・Cal.321：手巻き / 17石 / 18,000振動/時 / 1946年導入 / コラムホイール式

レマニア1873もコラムホイール式の手巻きクロノグラフキャリバーとして知られています。オメガ スピードマスタープロフェッショナルに採用されたほか、ジンのクロノグラフナビゲーションなど他ブランドへも供給されました。

━━━━━━━━━━━━━━

バルジュー系キャリバー ― 1960年代クロノグラフの標準

バルジューはスイスのル・バ・デュ・ピュイを拠点とするエブーシュメーカーで、ホイヤー・ユニバーサルジュネーブ・ブライトリングなど多くのクロノグラフブランドへキャリバーを供給しました。

バルジュー72は1960年代のクロノグラフに幅広く採用されたコラムホイール式キャリバーです。ホイヤーのオータヴィアをはじめ、さまざまなブランドの3レジスタークロノグラフに搭載されています。コラムホイール式を採用しているため、操作感の滑らかさはレマニア系と共通した特徴を持ちます。

バルジュー72Cはバルジュー72にトリプルカレンダー機能を追加したバリエーションで、ギャレットのトリプルカレンダークロノグラフなどに搭載されました。

・バルジュー72：コラムホイール式 / ホイヤー（オータヴィア等）、多数ブランド採用
・バルジュー72C：コラムホイール式 / トリプルカレンダー付き / ギャレット等採用

━━━━━━━━━━━━━━

ETA系キャリバー ― カム式で量産化を実現

ETAはスイスのグレンヒェンを拠点とするエブーシュメーカーで、現在はスウォッチグループ傘下に属します。その代表的なクロノグラフキャリバーがオメガ Cal.861で、手巻き・17石・21,600振動/時という仕様で1968年に導入されました。

Cal.861の最大の特徴は、コラムホイール式に代わってカム式クロノグラフ制御を採用した点です。偏心した板カム部品でレバーを切り替えるシンプルな構造により、製造コストを抑えながら量産に対応できます。振動数も18,000振動/時から21,600振動/時に引き上げられています。

・Cal.861：手巻き / 17石 / 21,600振動/時 / 1968年導入 / カム式

1968年にオメガがスピードマスタープロフェッショナルの搭載キャリバーをCal.321からCal.861に変更したのは、コスト面と生産効率の改善が主な背景でした。Cal.861はカム式であっても長年にわたり高い信頼性を維持し続けました。

━━━━━━━━━━━━━━

3社キャリバーの比較

・Cal.321（レマニア系）：17石 / 18,000振動/時 / コラムホイール式 / 1946年導入
・レマニア1873（レマニア）：コラムホイール式 / オメガ スピードマスター等に搭載
・バルジュー72（バルジュー）：コラムホイール式 / ホイヤー等多数ブランドに搭載
・バルジュー72C（バルジュー）：コラムホイール式 / トリプルカレンダー付き
・Cal.861（ETA系）：17石 / 21,600振動/時 / カム式 / 1968年導入

コラムホイール式を採用したレマニア系・バルジュー系は操作感の滑らかさで評価され、カム式のETA系は量産性・整備性の高さが強みです。いずれも実用上の信頼性は十分に証明されており、供給元の違いがモデルの性格を読み解く重要な視点になります。
