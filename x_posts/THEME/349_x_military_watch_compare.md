# 軍用（ミリタリー）時計 IWC・ロレックス・チューダー実例比較

ミリタリーウォッチは「軍が特定の仕様を指定し、官給品として支給した時計」を指します。同じリファレンスでも、軍用刻印のある個体がコレクター間で特別に扱われるのには、明確な理由があります。

今回はIWC マーク11、ロレックス サブマリーナ初期型、チューダー サブマリーナの3モデルを比較し、軍用供給の背景・仕様の違い・刻印の読み方を整理します。

なぜ軍は特別仕様の時計を発注したのか

航空作戦ではコックピットの計器類が強い磁場を発生させるため、一般的なムーブメントでは精度が保てませんでした。潜水作戦では水圧への対応が必須で、民間向けの防水性能では不十分でした。

こうした環境要件から、各国軍は時計メーカーに独自仕様を要求しました。耐磁性・防水性・視認性・時刻同期機能を満たしたモデルのみが官給品として採用され、支給番号やマーキングを刻印されたうえで支給されました。

IWC マーク11 — RAF向けパイロットウォッチ

IWC マーク11（Mk.XI）は1940年代後半から1950年代にかけてイギリス空軍（RAF）とオーストラリア空軍（RAAF）に納入されたモデルです。

主な仕様：
・ムーブメント：Cal.89（ハック機能付き手巻き）
・ケース径：36mm ステンレス 二重構造
・文字盤：ブラックマットダイヤル

Cal.89のハック機能（リューズ引き出し時に秒針停止）は軍用モデル限定の仕様で、パイロットが時報に合わせて秒単位で時刻を同期するために不可欠な機構でした。二重構造ケースは航空計器の磁場からムーブメントを守るための耐磁設計です。

RAF向けの識別ポイント：
・文字盤にブロードアロー（英国政府所有を示す矢印マーク）
・文字盤にTマーク
・裏蓋に「6B/346」（航空機器カテゴリの軍用リファレンス）
・ムーブメントにもブロードアロー刻印

RAAF向けは文字盤上のブロードアロー・Tマークが入らない仕様で、RAF向けとの見分けは文字盤マーキングの有無が最大の判断基準です。

ロレックス サブマリーナ初期型 — ダイバーズウォッチと軍用供給

1953年に登場したサブマリーナは、民間・プロフェッショナル向けとして発表されましたが、英国海軍等にも供給されました。

初期3リファレンスの特徴：

Ref.6204（1953〜1954年頃）
・Cal.A260（一方向自動巻き、18,000振動/時）
・ケース径37mm、スモールクラウン（5.3mm）
・防水100m（後に200m認証）
・ギルトダイヤル、初期にはハニカム文字盤あり

Ref.6536（1956〜1958年頃）
・Cal.1030（ロレックス初の双方向自動巻き、25石）
・ケース径37mm、スモールクラウン
・防水100m

Ref.6538（1955〜1959年頃）
・Cal.1030（双方向自動巻き）
・ケース径38mm、ビッグクラウン（8mm）
・防水200m
・後に「ジェームズ・ボンドサブマリーナ」の呼称

ビッグクラウンのRef.6538はダイバーがグローブ着用のまま操作できる実用設計で、軍用ダイバー向けの要求を意識した仕様です。軍用個体の識別は裏蓋の支給番号刻印の有無が主な手がかりとなります。

チューダー サブマリーナ — ロレックスと同等のケースで

チューダーはロレックス創設者ハンス・ウィルスドルフが設立したディフュージョンブランドです。ヴィンテージのチューダー サブマリーナの裏蓋には「ORIGINAL OYSTER CASE BY ROLEX GENEVA」と刻印されており、ロレックスと同じオイスターケースを使用していたことが明示されています。

ロレックスと同等のオイスターケース防水性能を持ちながら、ムーブメントにETA製キャリバーを採用してコストを抑えたことが、軍調達での採用理由のひとつでした。

代表Ref.の仕様：

Ref.7928（デカバラ・1960年代前半〜半ば）
・Cal.390（バタフライローター搭載のETA製自動巻き）
・ケース径40mm、ブラックベゼル（ロング5）
・文字盤：大きな薔薇ロゴ（デカバラ）

Ref.7016（小バラ・1960年代後半〜1970年代前半）
・ETA製自動巻き
・ケース径40mm
・文字盤：小さな薔薇ロゴ（小バラ）

3モデル横断比較

用途別の設計差が明確です。

マーク11（パイロット用）：
・耐磁性が最優先 → 二重ケース構造
・時刻同期が最優先 → ハック機能付き手巻き
・ケース径36mm（コンパクトな設計）

サブマリーナ系（ダイバー用）：
・防水性が最優先 → オイスターケース構造
・水中操作を意識 → ビッグクラウン（Ref.6538）
・視認性重視 → ブラック文字盤・夜光インデックス

チューダー サブマリーナ（コスト効率重視）：
・ケースはロレックスと同等のオイスターケース
・ムーブメントはETA製で調達コストを抑制
・40mmケースで視認性を確保

軍用刻印の読み方と偽造対策

ブロードアロー（英国軍）
英国政府所有を示す矢印状のマーク。IWC マーク11では文字盤・ムーブメントの両方に刻印が確認されます。

6B/346（RAF品番）
「6B」が航空機器カテゴリコード、「346」が品番。裏蓋への刻印がRAF官給品の証明となります。

支給番号（海軍等）
ロレックス等の英国海軍向け個体では裏蓋に支給番号が刻印されている場合があります。個体管理・部隊配布を示す記録です。

偽造・後刻印への確認ポイント：
・刻印の深さと均一性（オリジナルは機械的で均一、後刻印は不均一なことが多い）
・刻印位置が同Ref.の軍用モデルの標準パターンと一致しているか
・刻印の年代と時計全体の経年変化が整合しているか

コレクターとしての評価視点

軍用刻印のある個体が同Ref.の民間個体より評価される理由は「用途の明確さ」と「歴史的文脈」です。

軍調達品には仕様書があり、要求性能を満たすことが製造条件でした。支給記録と照合可能な個体は、単なる時計ではなく歴史的資料としての側面を持ちます。

オリジナリティの観点では：
・文字盤・ムーブメント・ケース・ブレスレットが同一個体由来であること
・文字盤リダン（再塗装）がなく、軍用マーキングが残っていること
・夜光の経年変化が均一で、後塗りの痕跡がないこと

以上が高評価個体の条件として挙げられます。
