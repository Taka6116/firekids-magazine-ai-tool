キングセイコー vs グランドセイコー -- 諏訪と亀戸、二つの頂点を知る

1960年に諏訪精工舎からグランドセイコー、翌1961年に第二精工舎（亀戸工場）からキングセイコー。同じセイコーの名を冠しながら、製造拠点も開発思想も異なる二つの高級ライン。その違いを世代ごとに整理します。

━━━━━━━━━━━━━━

[社内競争 -- 諏訪精工舎 vs 第二精工舎]

グランドセイコーは諏訪精工舎が「最高精度・最高仕上げ」を掲げて開発。キングセイコーは第二精工舎が「高品質かつ実用的」を標榜して独自に展開しました。

この競争がセイコー全体の技術水準を引き上げる原動力となっています。

[初代モデルの違い]

GS 1st（1960年）は14KGF（14金張り）ケースにCal.3180（25石、18,000振動/時）を搭載。製造期間わずか3年間のフラッグシップです。

KS 1st（1961年）はステンレスケース中心で、裏蓋には「楯メダリオン」が刻まれています。金張りではなくSSで勝負した実用路線が初代から明確でした。

[1960年代後半 -- ハイビート競争]

1967年、44GS（Cal.4420、25石、36,000振動/時）が亀戸初のグランドセイコーとして登場。翌1968年には45GS（Cal.4522）がGS規格の完成形・日差プラスマイナス2秒を達成しました。18金無垢ケース（Ref.4522-8010）も展開され、「スイス以上のモノづくり」を体現しています。

同時期のキングセイコー 44KS（Ref.4402-8000）は、ハック機能とカレンダー早送りを備えた実用モデル。精度競争よりも日常使いの機能充実に注力しました。

[自動巻きと最終世代]

GS側は62GS（Cal.6245、1967年）で自動巻きを初導入し、61GS（Cal.6145/6146、36,000振動/時）、56GS（Cal.5641/5645/5646）へと進化。

KS側は56KS（Cal.5625/5626）でワンピースケースの自動巻きハイビートに到達し、1975年の52KSスペシャル（Ref.5256-8000）でシリーズ終了。製造期間わずか8か月の最終モデルです。

[総合比較]

・製造拠点：GS=諏訪精工舎 / KS=第二精工舎
・方向性：GS=最高精度・最高仕上げ / KS=高品質かつ実用的
・ケース素材：GS=14KGF、キャップゴールド、18KYG / KS=SS中心
・精度規格：GS=GS規格（独自基準） / KS=一部クロノメーター仕様
・裏蓋：GS=ライオン・GS / KS=楯（1st）→獅子（44KS以降）
・生産期間：GS=1960年〜（復活含む） / KS=1961〜1975年

━━━━━━━━━━━━━━

グランドセイコーは「スイスを超える精度」を、キングセイコーは「品質と実用のバランス」を追求しました。この二拠点の競争が36,000振動/時のハイビートやセイコースタイルを生み出し、セイコーを世界水準へ押し上げたのです。
