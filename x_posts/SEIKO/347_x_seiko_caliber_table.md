セイコー ヴィンテージ 1960〜70年代 Cal.番号総まとめ

1960〜70年代のセイコーには、グランドセイコー・キングセイコーの精度競争路線からダイバーズ・クロノグラフのスポーツ路線まで、多種多様なキャリバーが存在します。

リファレンス番号の先頭4桁がキャリバー番号に対応するセイコーの命名規則を理解すると、ヴィンテージセイコーの仕様確認が格段にしやすくなります。

グランドセイコー 手巻きキャリバーの系譜

GS手巻きキャリバーは8年間で劇的に進化しました。

1960年のCal.3180（57GS 1stモデル）は18,000振動/時・25石のロービート仕様で、精度規格は+12/-3秒/日。1964年のCal.430/5722（57GS 2ndモデル）も同じロービートですが、1967年のCal.4420（44GS）でついに36,000振動/時のハイビートに移行します。

1968年のCal.4520/4522（45GS）では±2秒/日という精度に到達。COSCクロノメーター規格（-4/+6秒/日）を明確に上回る水準で、GS規格の完成形と位置づけられます。

手巻きキャリバー早見表
・Cal.3180 → 57GS 1st（Ref.J14070）　18,000振動/時
・Cal.430 → 57GS 2nd前期（Ref.5722-9000）　18,000振動/時
・Cal.5722 → 57GS 2nd後期（Ref.5722-9011）　18,000振動/時
・Cal.4420 → 44GS（Ref.4420-9000）　36,000振動/時
・Cal.4520 → 45GS（Ref.4520-8000）　36,000振動/時
・Cal.4522 → 45GS（Ref.4522-8000/8010）　36,000振動/時

グランドセイコー 自動巻きキャリバーの系譜

GS初の自動巻きはCal.6245/6246（62GS）で、1967年に登場しました。ロービート（18,000振動/時）で、Cal.6246はデイデイト付きです。

翌1968年にはCal.6145/6146（61GS）が登場し、自動巻きでも36,000振動/時のハイビートを達成。1970年のCal.5641/5645/5646（56GS）ではハイビートを維持しながらさらなる薄型化が実現されています。

・Cal.6246 → 62GS（Ref.6246-9001）　18,000振動/時・デイデイト
・Cal.6145 → 61GS（Ref.6145-8000）　36,000振動/時・デイト
・Cal.6146 → 61GS（Ref.6146-8000）　36,000振動/時・デイデイト
・Cal.5641 → 56GS（Ref.5641-7000）　36,000振動/時・ノンデイト
・Cal.5645 → 56GS（Ref.5645-7010）　36,000振動/時・デイト
・Cal.5646 → 56GS（Ref.5646-7005）　36,000振動/時・デイデイト

キングセイコー系キャリバー

キングセイコーは第二精工舎（亀戸）がGSと精度を競ったモデルです。

44KS（Cal.4402）は36,000振動/時の手巻きハイビートで、Ref.4402-8000が代表的なリファレンス。56KS（Cal.5625/5626）はGSの56世代と同時期ながら、振動数は28,800振動/時とGSより低めに設定されています。

52KS（Cal.5256）はキングセイコー最終モデルで、1975年のわずか8ヶ月程度という短期間しか生産されませんでした。Ref.5256-8000として知られ、デイデイト表示付きの自動巻きキャリバーです。

キングセイコーキャリバー早見表
・Cal.4402 → 44KS（Ref.4402-8000）　手巻き・36,000振動/時
・Cal.5625 → 56KS（Ref.5625-7000）　自動巻き・28,800振動/時
・Cal.5626 → 56KS（Ref.5626-7000）　自動巻き・デイデイト
・Cal.5256 → 52KS（Ref.5256-8000）　自動巻き・KS最終モデル

ダイバーズ・スポーツ系キャリバー

セイコーの1960〜70年代ダイバーズには、Cal.6105系とCal.6306系という二世代が存在します。

Cal.6105A（2ndダイバー前期・Ref.6105-8000）とCal.6105B（後期・Ref.6105-8110）は、植村直己氏が着用した「ウエムラダイバー」として有名です。前期はリューズロック機構なし、後期は同機構が追加されています。

Cal.6306A（3rdダイバー・Ref.6306-7001）は1976〜1978年頃のモデルに搭載されたキャリバーで、手巻き機能を持たない自動巻き専用です。

クロノグラフのCal.6138は「5スポーツ スピードタイマー」用の自動巻きクロノグラフで、12時位置にリューズとプッシャーを配置した「ブルヘッド（茶馬）」（Ref.6138-0040）の搭載機として広く知られています。

Cal.6217はワールドタイムモデル（Ref.6217-7000・Ref.6217-7010）の自動巻きキャリバーで、1964年東京オリンピック記念モデルや1966年アジア大会記念モデルに採用されました。
