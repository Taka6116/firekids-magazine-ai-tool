# ロンジン ヴィンテージ入門｜スイス実用時計の真髄

ロンジン（LONGINES）は1832年創業、スイスのサン＝ティミエを拠点とする時計メーカーです。自社一貫生産によるキャリバー開発の歴史を持ち、ヴィンテージ時計の世界で精度・信頼性・バリエーションの豊富さという観点から際立った存在感を持っています。

ロレックスやオメガほど知名度は高くありませんが、スイス実用時計としての完成度は引けを取りません。これからヴィンテージウォッチを始める方に向けて、ロンジンを選ぶ理由・モデルの選び方・キャリバーの基礎知識を整理します。

なぜロンジンが入門者に向いているのか

ロンジンのヴィンテージが入門者にとって選びやすい理由は3つあります。

・自社キャリバーの蓄積が豊富で、モデルごとにキャリバーと特徴が明確に分かれている
・コンクエスト・フラッグシップなどの定番モデルは市場での流通量が比較的多い
・派手な演出より実用性・精度を重視した設計が多く、適切なメンテナンスで現在も使い続けられる個体が多数存在する

最初の1本として向いているモデル

【コンクエスト（Conquest）】

1950年代にオメガ コンステレーションへの対抗モデルとして誕生したスポーツ・ドレス兼用ライン。コンクエスト専用に開発されたCal.291（自動巻き）を搭載したモデルが多く、ローター軸受けにルビーを使用した精度と耐久性重視の設計が特徴です。

ケースはステンレス製のラウンドが主流で、ツイストラグ（ラグ部分にひねりを持たせた造形）がコンクエストを見分ける際の目印になります。ケースサイズは35mm前後（リューズ別）が標準的で、現代の腕に着けても違和感の少ないサイズ感です。流通量が多いため選択肢が広く、入門者が最初に比較検討するモデルとして定番です。

【フラッグシップ（Flagship）】

ロンジンのドレスラインとして展開されたモデルで、手巻きCal.30L系を搭載した個体が多く存在します。Cal.30Lはスモールセコンドを持つシンプルな手巻きムーブメントで、部品点数が少なくメンテナンス性に優れた実用機です。

文字盤はシルバー・ブルーグレー・ブルーなどのバリエーションがあり、ドレッシーな雰囲気を重視したい方に向いています。毎日自分でゼンマイを巻く行為が時計との付き合いを深めるきっかけになることも多く、はじめて手巻きヴィンテージを持つ方にも適したモデルです。

【ウルトラクロン（Ultra-Chron）】

1960年代後半に登場したハイビート自動巻きウォッチ。Cal.431（36,000振動/時・毎秒10振動）を搭載し、NASAのアポロ計画において公式時計の最終選定まで残ったムーブメントのひとつとして知られています。

ラウンドケース（Ref.7885-4：35.5mm、Ref.8301：37mm）やCラインケース（Ref.8317 1：35mm）など複数のケース形状が存在し、シルバーダイヤルやクロスラインのグレーダイヤルなど文字盤バリエーションも豊富です。コンクエストより流通量が少ないため、じっくり個体を選ぶ姿勢が必要ですが、スイスヴィンテージとしての完成度は高いモデルです。

Cal.30L系（手巻き）とCal.291（自動巻き）── まず知るべき2系統

ロンジンのヴィンテージキャリバーは多数ありますが、入門者がまず把握しておくべきは以下の2系統です。

・Cal.30L系（手巻き）
　→ フラッグシップ等に搭載。スモールセコンド。部品点数が少なくシンプルな構造。
　→ 毎日の巻き上げが必要だが、メンテナンス面での安心感がある。

・Cal.291（自動巻き）
　→ コンクエスト専用設計。センターセコンド。ローター軸受けにルビーを使用。
　→ 着用中に自動巻き上げが行われるため日常使いしやすい。

どちらもロンジンを代表するヴィンテージキャリバーとして評価が安定しており、メンテナンス実績も多い系統です。

最初の1本を選ぶ3つのポイント

1. 文字盤の状態を確認する
　→ サンバースト仕上げの輝きが保たれているか、変色・シミ・剥離がないか。リダン（再印刷）の有無も確認すべき項目です。

2. 搭載キャリバーを確認する
　→ モデル名だけでなくキャリバー番号を確認することで、時代・特性・メンテナンスの目安が立てられます。コンクエストはCal.291、フラッグシップはCal.30L系が基本です。

3. 純正ブレスレットと付属品の有無を確認する
　→ 純正ブレスレット・箱・保証書・製造証明書（アーカイブ）が揃った個体は来歴の証拠になります。コンクエストの純正ブレスレット付き個体は市場評価が高い傾向があります。
