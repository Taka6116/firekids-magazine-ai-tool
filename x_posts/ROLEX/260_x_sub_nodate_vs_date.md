ロレックス サブマリーナ「ノンデイト」と「デイト付き」の違いとは

サブマリーナには日付表示のないノンデイトと、日付表示付きのデイトという2つの系統があります。サイクロップレンズの有無だけでなく、搭載キャリバー、クロノメーター認定、ケース構造まで両者の違いは多岐にわたります。

━━━━━━━━━━━━━━

代表モデルの仕様比較

ヴィンテージ世代の代表であるRef.5513（ノンデイト）とRef.1680（デイト付き）を比較すると、違いが明確に見えてきます。

・Ref.5513：Cal.1520搭載、非クロノメーター、ハック機能なし、製造期間約27年
・Ref.1680：Cal.1570搭載、クロノメーター認定、ハック機能あり（1972年頃以降）、サブマリーナ初の日付表示

現代寄りの世代ではRef.14060（Cal.3000、非クロノメーター）とRef.16610（Cal.3135、クロノメーター認定）という組み合わせになります。Ref.16610は前モデルRef.168000のCal.3035からCal.3135に変更された、20年超のロングセラーです。

━━━━━━━━━━━━━━

キャリバーの違い

Cal.1520とCal.1570はともに26石・18,000振動/時の1500系キャリバーですが、精度規格が異なります。

Cal.1520はフラットヘアスプリングとレギュレーター方式で非クロノメーター。Cal.1570はブレゲヒゲとマイクロステラ精密調整機構を備えたクロノメーター仕様です。

3000系世代ではCal.3135（デイト付き）が31石・28,800振動/時。ノンデイト側のCal.3130はCal.3135から日付機構を省いた構成で、Ref.14060Mからクロノメーター認定を取得しています。

━━━━━━━━━━━━━━

外装上の違い

最も分かりやすいのがサイクロップレンズの有無。デイト付きでは3時位置の日付窓に2.5倍の拡大レンズが付きます。ノンデイトは風防がフラットです。

ダイヤルレイアウトも異なります。ノンデイトは日付窓がないため左右対称のすっきりとしたデザイン。デイト付きは3時位置のインデックスが省略・短縮され、日付窓が配されます。

ケース厚はデイト付きがわずかに厚い傾向にありますが、装着感に大きな差が出るほどではありません。

━━━━━━━━━━━━━━

リファレンスの系譜

ノンデイト：Ref.5512 → Ref.5513 → Ref.14060 → Ref.14060M → Ref.114060
デイト付き：Ref.1680 → Ref.16800 → Ref.168000 → Ref.16610 → Ref.116610

注目すべきは、ノンデイトがサファイアクリスタルを採用したのはRef.14060（1990年頃）からですが、デイト付きはRef.16800（1979年頃）で既に採用しており、約10年の差があります。

━━━━━━━━━━━━━━

ノンデイトは日付を合わせる必要がなく、複数の時計をローテーションする方には使いやすい仕様です。デイト付きはビジネスシーンでの実用性に優れます。ヴィンテージ世代ではノンデイト=実用機、デイト付き=上位機という位置づけが明確でしたが、Ref.14060M以降その差は解消されています。
