# X投稿：ロレックス デイトジャスト フルーテッドベゼル vs スムースベゼル

---

ロレックス デイトジャストを選ぶとき、どのベゼルを選ぶかで印象が大きく変わります。

デイトジャストにはフルーテッドベゼル（縦溝入り）、スムースベゼル（無地）、エングレーブドベゼル（刻み模様）といった種類があり、それぞれ対応するRef.番号も異なります。「デイトジャストといえばフルーテッド」というイメージが強い一方で、スムースベゼルやエングレーブドベゼルを好むコレクターも多く、ベゼルの違いはデイトジャスト選びの重要な軸になります。

フルーテッドベゼルとは

フルーテッドベゼルは、ベゼル表面に均等な縦溝が刻まれたデザインです。デイトジャスト誕生の1945年当時から採用されており、ロレックスの「顔」として定着しています。

代表的なRef.と仕様は以下のとおりです。

・Ref.6605（1950年代後半）：イエローゴールドのフルーテッドベゼル、細かな溝が入った初期タイプ、Cal.1065搭載
・Ref.1601（1960〜1981年頃）：ホワイトゴールドのフルーテッドベゼル、Cal.1570（26石・18,000振動/時）搭載、コンビや金無垢バリエーションあり
・Ref.16014（1979〜1988年頃）：ホワイトゴールドのフルーテッドベゼル、Cal.3035（27石・28,800振動/時）搭載、サファイアクリスタル風防
・Ref.16234（1988年頃〜2000年代半ば）：ホワイトゴールドのフルーテッドベゼル、Cal.3135（31石・28,800振動/時）搭載

フルーテッドベゼルは光の当たる角度で表情が変わり、ホワイトゴールドの素材感と縦溝の組み合わせがドレスウォッチとしての格を高めます。ヴィンテージ個体ではベゼルの溝エッジの立ち具合が状態の指標にもなります。

━━━━━━━━━━━━━━

スムースベゼルとは

スムースベゼルは装飾のない平滑なベゼルです。シンプルで飾り気のない外観が特徴で、文字盤の色や素材を問わず幅広い組み合わせに対応します。

デイトジャストにおけるスムースベゼルの代表Ref.はRef.1600です。

・Ref.1600（1959〜1980年頃）：SSスムースベゼル、Cal.1560（初期型）またはCal.1570搭載。コンビモデル（SS×YG）ではイエローゴールドのスムースベゼルが採用されたバリエーションも存在します。

Ref.1600はフルーテッドベゼルのRef.1601に比べると流通量が少なく、シンプルなデザインを好むコレクターや、デイトジャストを独自のスタイルで楽しみたい方に選ばれることが多いモデルです。

セミヴィンテージ以降では、Cal.3135搭載のRef.16200がスムースベゼルにサファイアクリスタル風防を組み合わせた仕様で展開されています。

━━━━━━━━━━━━━━

エングレーブドベゼル（エンジンターンドベゼル）

エングレーブドベゼルは、ベゼル面に細かな刻み模様（ローレット）が施されたタイプです。フルーテッドとスムースの中間的な印象を持ちます。

・Ref.1603（1960〜1970年代後半）：ステンレスのエングレーブドベゼル、Cal.1570搭載

Ref.1603はRef.1601とケースサイズ（36mm）やキャリバー（Cal.1570）を共有しながら、ベゼルの素材と仕上げが異なります。全面ステンレスの仕様でオイスターブレスレットとの相性が良く、スポーティかつ落ち着いた印象があります。

━━━━━━━━━━━━━━

Ref.とベゼルの対応まとめ

・Ref.6605（1950年代後半）：YGフルーテッド / Cal.1065
・Ref.1600（1959〜1980年頃）：SSスムース（コンビはYG） / Cal.1560・1570
・Ref.1601（1960〜1981年頃）：WGフルーテッド / Cal.1560・1570
・Ref.1603（1960〜1970年代後半）：SSエングレーブド / Cal.1570
・Ref.16014（1979〜1988年頃）：WGフルーテッド / Cal.3035
・Ref.16234（1988〜2000年代半ば）：WGフルーテッド / Cal.3135
・Ref.16200（1989〜2000年代）：SSスムース / Cal.3135

━━━━━━━━━━━━━━

どのベゼルを選ぶか

フルーテッドベゼルはデイトジャストの正統派スタイルを求める方、フォーマルシーンで時計の格を求める方に向いています。ヴィンテージのRef.1601であれば、ホワイトゴールドのフルーテッドとジュビリーブレスレットを組み合わせた王道の組み合わせが楽しめます。

スムースベゼルは、飾り気のないシンプルな外観を好む方や、デイトジャストを独自のスタイルで選びたい方に向いています。Ref.1600のヴィンテージは流通量が少なく、独自路線を好むコレクターからも注目されています。

エングレーブドベゼルのRef.1603は、フルーテッドとスムースの中間的な存在として、デイトジャスト入門から本格的なコレクションまで幅広い層に選ばれてきたモデルです。

ベゼルの違いを理解することで、デイトジャスト選びの視点が具体的になります。どのRef.を選ぶかを考える際、ベゼルタイプを出発点にするのは有効なアプローチです。
