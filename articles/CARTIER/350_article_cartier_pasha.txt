# カルティエ パシャ・ドゥ・カルティエ 初代解説｜FIRE KIDS Magazine

カテゴリ：カルティエ

---

カルティエ パシャは、ラウンドケースと防水性能を備えたスポーツウォッチとして展開されたモデルです。タンクやサントスが角型ケースを特徴とするのに対し、パシャは丸いケース形状を採用し、ねじ込みリューズによる防水仕様がひとつの大きな特徴となっています。

この記事では、パシャCを中心に初期型の仕様・キャリバー・文字盤バリエーション、およびシータイマーやグリッドといった派生モデルについて解説します。すでにカルティエを所有している方、あるいはパシャへの関心を持つ時計愛好家の参考になれば幸いです。

## パシャCの基本仕様

パシャC（Pasha C）は、カルティエのスポーツウォッチラインを代表するモデルです。ラウンドケースにねじ込みリューズを組み合わせた防水仕様が特徴で、アクティブなシーンにも対応するカルティエとして位置づけられています。

主要リファレンスはRef.2324で、ケースサイズは35mm（リューズ別）のステンレスケースとなっています。ムーブメントはETA 2892A2ベースの自動巻きキャリバーを搭載しており、片方向巻き上げのローターを持つ信頼性の高いムーブメントです。

| 項目 | 仕様 |
|------|------|
| リファレンス | Ref.2324 |
| ケース素材 | ステンレス |
| ケースサイズ | 35mm（リューズ別） |
| ムーブメント | 自動巻き / Cal.ETA2892A2 |
| 防水仕様 | ねじ込みリューズ |

## パシャCの初期型について

パシャCには初期型と呼ばれる仕様が存在します。初期型はRef.2324をベースとしており、シルバーダイヤルが代表的な文字盤として確認されています。製造年代は1980年代から1990年代にかけてのモデルが存在しており、1980年代製のパシャCは流通数が少なくなっています。

初期型と後期型の主な差異として、ブレスレットの仕様変更が挙げられます。初期型は純正ブレスレットが付属する個体と、革ベルト仕様の個体が混在しており、ブレスレット付きの初期型は確認できる個体数が限られています。

## 文字盤バリエーション

パシャCの文字盤バリエーションは多岐にわたります。ホワイトダイヤルが最も一般的なバリエーションですが、ほかにもシルバー、ブラック、ブラックアラビア、ピンク、カッパーなど複数のカラーが展開されています。

| 文字盤 | 備考 |
|--------|------|
| ホワイトダイヤル | 最も流通量の多い標準的なバリエーション |
| シルバーダイヤル | 初期型に多く見られる仕様 |
| ブラックダイヤル | ラグジュアリーな印象を持つ人気バリエーション |
| ブラックアラビアダイヤル | アラビア数字インデックスのブラック文字盤 |
| ピンクダイヤル | 落ち着いたピンクカラーの文字盤 |
| カッパーダイヤル | Ref.2334に確認されるカッパーカラー |

### トリチウム夜光について

1990年代製のパシャCには、夜光塗料にトリチウムを使用した個体が確認されています。トリチウム夜光は経年により変色することが一般的で、インデックスや針に黄変・クリーム変色が見られる個体が多くあります。トリチウム表記の確認はヴィンテージとしての時代感を知るうえでの指標となります。

## K18YGのパシャC

パシャCにはステンレスモデル以外に、18金イエローゴールドケースのバリエーションも存在します。Ref.W30134H9がK18YGのパシャCとして確認されており、純正18K尾錠が付属する個体も存在します。ステンレスケースとは異なる素材感と重量感があり、ドレッシーな方向での着用が想定された仕様です。

## パシャCの派生モデル

### パシャシータイマー

パシャシータイマー（Pasha Seatimer）は、パシャCよりも大径のケースを持つスポーティなバリエーションです。ケースサイズは38mm（リューズ別）で、防水仕様としてリューズカバーが採用されています。ダイバーズウォッチ的な方向性を持つモデルで、回転ベゼルを備えたバリエーションも存在します。1990年代製の個体が多く確認されており、ムーブメントは自動巻きです。

シータイマーにはGMTパワーリザーブ機能を持つシータイマーGMTも存在します。38mmケースにパワーリザーブインジケーターを備えた複合機能モデルで、2001年販売ギャランティーが確認されています。

| 項目 | 仕様 |
|------|------|
| ケースサイズ | 38mm（リューズ別） |
| ムーブメント | 自動巻き |
| 防水仕様 | リューズカバー |
| バリエーション | 通常型、GMT＋パワーリザーブ型 |

### パシャグリッド

パシャグリッド（Pasha Grid）は、文字盤にグリッド（格子）模様を採用したバリエーションです。Ref.W3104055（別表記：Ref.2379）が確認されており、ホワイトダイヤルとのバリエーションが存在します。1990年代から2000年代にかけての個体が確認されています。

### パシャCメリディアン

パシャCメリディアンは、GMT機能を搭載したパシャCの派生モデルです。Ref.2377およびRef.W31078M71が確認されており、2つのタイムゾーンを表示できる機能を備えています。シルバーダイヤルとブラックダイヤルのバリエーションが確認されており、1990年代から2000年代にかけて製造されました。

ビッグデイト表示を組み合わせたパシャCメリディアン GMTビッグデイトも存在し、Ref.2377のブラックダイヤル仕様が2000年代製として確認されています。

### パシャC ビッグデイト

通常のデイト表示より大きな数字で日付を表示するビッグデイト機構を持つバリエーションです。Ref.2475として確認されており、2000年代製のブラックダイヤル仕様が存在します。2004年発行のギャランティーが確認されている個体もあります。

### 特別仕様モデル

2002年製のパシャとして、Ref.S091が確認されています。FIFAワールドカップを記念した世界70本限定のモデルで、18金ホワイトゴールドケースを採用した特別仕様です。製造本数の少なさから、コレクター性が高い個体として位置づけられます。

## ヴィンテージのパシャCを選ぶポイント

### ねじ込みリューズとキャップの確認

パシャCの大きな特徴であるねじ込みリューズには、チェーン付きのセーフティキャップが付属するモデルがあります。チェーンとキャップが揃った状態かどうかは、パシャCのオリジナリティを確認するうえでの重要なポイントです。

### ブレスレットの状態

パシャCの純正ブレスレットは、使用による伸びが生じる場合があります。ブレスレット付きで販売される場合は、リンクの状態と全体的な伸び具合を確認することが重要です。

### 初期型と後期型の違い

初期型（初期製造バリエーション）と後期型では、ブレスレットデザインや文字盤の細部に差異が見られることがあります。特にシルバーダイヤルの初期型はヴィンテージとしての特徴がより強く出やすいモデルです。

### ギャランティーの有無

1990年代から2000年代製のパシャCでは、ギャランティー（保証書）付きの個体が流通しています。発行年がわかるギャランティーは製造年代を判断する補助的な情報となります。

| パシャC カッパーダイヤル | 2334 | 35mm | 自動巻き |

## よくある質問（FAQ）

Q: パシャCの初期型と後期型はどのように見分けますか？

A: 初期型の特徴として、シルバーダイヤルを採用した個体が多い点が挙げられます。ブレスレットのデザインや、文字盤のインデックス・针の形状にも時代ごとの差異が見られます。Ref.2324は長期にわたって使用されたリファレンスのため、製造年代の判断にはギャランティーや専門書等の補助情報が参考になります。

Q: パシャCとパシャシータイマーの違いは何ですか？

A: ケースサイズと防水仕様が主な違いです。パシャCはケースサイズ35mm（リューズ別）でねじ込みリューズによる防水仕様ですが、パシャシータイマーは38mm（リューズ別）と大きく、リューズカバーを採用しています。シータイマーはよりスポーティなダイバーズ寄りの設計で、回転ベゼルを持つバリエーションも存在します。

Q: パシャCのムーブメントはどのようなものですか？

A: 確認されているパシャCのムーブメントはETA 2892A2ベースの自動巻きキャリバーです。ETA 2892A2はスリムな設計と信頼性の高さで知られる汎用キャリバーで、カルティエを含む多くの高級時計ブランドで採用されています。28,800振動/時の高振動設計で、日差±4〜5秒程度の実用精度を発揮します。

Q: パシャCの文字盤でどのバリエーションが確認されていますか？

A: ホワイト、シルバー（初期型に多い）、ブラック、ブラックアラビア、ピンク、カッパーなどのバリエーションが確認されています。ホワイトダイヤルが最も流通量が多く、ブラックダイヤルも1990年代から2000年代にかけて複数の個体が確認されています。

Q: パシャCにはゴールドケースのモデルはありますか？

A: はい、Ref.W30134H9として18金イエローゴールドケースのパシャCが確認されています。1990年代製のモデルで、純正18K尾錠が付属する個体も存在します。

## まとめ

カルティエ パシャCは、ラウンドケースとねじ込みリューズ防水を特徴とする自動巻きスポーツウォッチです。主要リファレンスRef.2324を軸に、シルバーダイヤルの初期型から多彩な文字盤バリエーション、GMT機能を持つメリディアン、大径ケースのシータイマー、格子文字盤のグリッドまで、多くのバリエーションが展開されました。

ステンレスケースによる耐久性と、ETA 2892A2を搭載した実用的な自動巻き機構は、日常的な使用にも適したカルティエの一面を示しています。タンクやサントスとは異なるアプローチで作られたパシャCは、カルティエのコレクションの中でも独自の位置を占めるモデルです。
