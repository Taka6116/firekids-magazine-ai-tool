チューダー（TUDOR）のサブマリーナは、ロレックスのオイスターケースと王冠リューズを共有しながら、チューダー独自のロゴと仕様を組み合わせたダイバーズウォッチです。その中でも、1960年代前半から1970年代前半にかけて製造されたRef.7928とRef.7016は、文字盤に薔薇ロゴを配した「バラサブ」として知られ、チューダー サブマリーナの歴史を語るうえで欠かせない存在です。

本記事では、Ref.7928およびRef.7016の仕様と特徴、ロレックスとの関係、そして搭載されたムーブメントの変遷を、現物の確認情報をもとに解説します。チューダー サブマリーナの歴史に関心のある時計愛好家の方に向けた内容です。

---

## チューダー サブマリーナとロレックスの関係

### オイスターケースと王冠リューズの共有

チューダー サブマリーナは、ロレックスのオイスターケース（Oyster Case）を採用しています。裏蓋には「ORIGINAL OYSTER CASE BY ROLEX GENEVA」の刻印が確認でき、ケース構造においてロレックス サブマリーナとの共通性が裏付けられています。

ねじ込み式の王冠リューズもロレックス製のものが採用されており、ケース・リューズといった外装部品はロレックスと共通のパーツが使用されていました。これはチューダーがロレックスのディフュージョン（廉価ライン）として位置づけられていた時代の特徴であり、耐久性・防水性能の面ではロレックスと同等のオイスターケースの恩恵を受けていました。

### ムーブメントはETA製を採用

ロレックス サブマリーナがロレックス自社製ムーブメントを搭載するのに対し、チューダー サブマリーナのRef.7928・Ref.7016の時代はETA（Ebauches SA）製キャリバーを採用していました。これによりロレックスより低価格で提供しながら、ケースはロレックスと同等の品質を維持するというブランドポジションが実現されていました。

### ブレスレットの共有

初期のバラサブには、ロレックス製のステンレスブレスレットが装着されていました。リベットブレス（7206等）や3連ステンレスブレスが確認されており、ブレスレットにおいてもロレックスとのパーツ共有が行われていたことがわかります。

## Ref.7928（デカバラ サブマリーナ）の特徴

### モデル概要

Ref.7928は、チューダー サブマリーナにおいて「バラサブ」「デカバラ」の代名詞となっているリファレンスです。文字盤12時位置に大きな薔薇ロゴが配されており、ダイバーズウォッチの武骨なオイスターケースと薔薇ロゴの組み合わせが独特のコントラストを生み出しています。

1960年代前半から半ばにかけての製造が確認されており、チューダー サブマリーナの中でも最も古い世代のモデルのひとつです。

### Ref.7928のスペック

| 項目 | 仕様 |
|---|---|
| 製造年代 | 1960年代（1963年製、1964年製の個体が確認されている） |
| ムーブメント | 自動巻き / Cal.390（バタフライローター搭載） |
| ケースサイズ | 40mm |
| ケース素材 | ステンレス |
| ラグ幅 | 20mm |
| ベゼル | ブラック（ロング5ベゼル） |
| 裏蓋刻印 | ORIGINAL OYSTER CASE BY ROLEX GENEVA |
| ブレスレット | ロレックス製3連ステンレスブレス / リベットブレス |

### Cal.390のバタフライローター

Ref.7928に搭載されたCal.390は、バタフライローターと呼ばれる特徴的な形状の自動巻きローターを持つキャリバーです。蝶の羽のように左右に広がるローターの形状は、同時代のETA製キャリバーの中でも独特の存在感を持っています。このバタフライローターはCal.390の視覚的な特徴として、チューダー コレクターの間でも注目されています。

### ロング5ベゼル

Ref.7928のベゼルは「ロング5ベゼル」と呼ばれ、5分位置の目盛りが長く刻まれているのが特徴です。この仕様は1960年代初期のロレックス サブマリーナにも共通して見られるデザインで、同時代のロレックスとの共通性を示す要素のひとつです。ベゼルの退色（褪色）が進んだ個体では、独特の雰囲気を醸し出します。

### 文字盤の外周サークル

Ref.7928の文字盤外周には、分目盛り用のサークル（外周リング）が配されています。この外周サークルはロレックス サブマリーナ同時代のモデルとは若干異なる意匠で、チューダーとしての独自性を表す要素のひとつです。

## Ref.7016（小バラ サブマリーナ）の特徴

### モデル概要

Ref.7016は、1960年代後半から1970年代前半にかけて製造された、チューダー サブマリーナの「小バラ」世代のリファレンスです。文字盤12時位置の薔薇ロゴがデカバラ（Ref.7928）よりも小さくなり、より控えめな印象を与えるデザインになっています。

サフィックス「/0」が付くRef.7016/0が知られており、薔薇ロゴ世代のチューダー サブマリーナとしては最終世代に位置します。Ref.7016の後継として、1970年代後半以降は盾ロゴ（シールドロゴ）のモデルへと移行していきます。

### Ref.7016のスペック

| 項目 | 仕様 |
|---|---|
| 製造年代 | 1960年代後半から1970年代前半 |
| ムーブメント | 自動巻き / ETA製キャリバー |
| ケースサイズ | 40mm |
| ケース素材 | ステンレス |
| ベゼル | ブラック |
| 裏蓋刻印 | ORIGINAL OYSTER CASE BY ROLEX GENEVA |
| ブレスレット | ロレックス製巻きブレス等 |

### デカバラ（Ref.7928）との違い

| 項目 | Ref.7928（デカバラ） | Ref.7016（小バラ） |
|---|---|---|
| 製造年代 | 1960年代（前半〜半ば） | 1960年代後半〜1970年代前半 |
| 薔薇ロゴのサイズ | 大きい（デカバラ） | 控えめ（小バラ） |
| キャリバー | Cal.390（バタフライローター） | ETA製自動巻き |
| ケースサイズ | 40mm | 40mm |
| ベゼルカラー | ブラック | ブラック |
| ブレスレット | ロレックス製3連・リベットブレス | ロレックス製巻きブレス等 |
| 薔薇ロゴの意義 | デカバラ世代の幕開け | 薔薇ロゴ世代の最終世代 |

## チューダー サブマリーナのロゴ変遷

### 薔薇ロゴから盾ロゴへ

チューダーのロゴは時代によって変化しています。サブマリーナにおいては以下の変遷が確認されています。

| 時代 | ロゴ | 代表的なRef. |
|---|---|---|
| 1960年代前半まで | デカバラ（大きな薔薇） | Ref.7928 |
| 1960年代後半〜1970年代前半 | 小バラ（小さな薔薇） | Ref.7016 |
| 1970年代後半以降 | 盾ロゴ（シールド） | Ref.7021以降 |

薔薇ロゴはイングランドのチューダー王家の家紋に由来し、チューダーブランド創設期から使用されてきたものです。1970年代後半に盾ロゴへ移行したことで、薔薇ロゴを持つサブマリーナは「バラサブ」として、現在では特別なヴィンテージポジションを占めるようになりました。

### スノーフレーク世代への橋渡し

薔薇ロゴから盾ロゴへの移行とほぼ同時期に、チューダー サブマリーナではスノーフレーク針（通称：イカサブ）を採用した世代が登場しています。Ref.7928やRef.7016のメルセデス針仕様から、スノーフレーク針を採用したRef.7021（1968年頃よりCal.2484搭載）への展開は、チューダー サブマリーナのデザインとムーブメントの両面での世代交代を示しています。

## バラサブを見極めるポイント

### 文字盤のオリジナル性

薔薇サブの中にはリダン（文字盤の再塗装）が施された個体が存在します。文字盤上の薔薇ロゴの印刷精度、夜光インデックス・針の経年変化の均一性は、オリジナル文字盤を判断するうえでの重要な手がかりとなります。コレクション目的で購入を検討する場合、文字盤のオリジナル性の確認は特に重要です。

### 夜光の経年変化

1960年代製のRef.7928では、文字盤インデックスと針の夜光が経年によって変色しています。均一に経年変化した夜光は、文字盤がオリジナルであることの手がかりとなります。また、針の夜光が塗り直されていないか、針自体が交換されていないかも確認すべき点です。

### ベゼルインサートの状態

ブラックベゼルのバラサブでは、経年によってベゼルインサートが退色し、グレーや褐色に変化した個体も存在します。この退色はヴィンテージとしての味わいの一部として評価されています。ベゼルが交換されていないかの確認も、購入時の重要なチェックポイントです。

### ブレスレットの状態

初期のRef.7928にはリベットブレスが装着された個体が存在します。ロレックス製のリベットブレスや3連ステンレスブレスがオリジナル状態で揃っている個体は、状態面での価値が高まります。後年に社外ブレスや別世代のブレスに交換されている個体も多いため、ブレスレットのオリジナル性も確認の対象となります。

## よくある質問

**Q:** Ref.7928のCal.390とはどのようなムーブメントですか？

**A:** Cal.390はETA製の自動巻きキャリバーで、バタフライローターと呼ばれる蝶の羽状のローターが特徴です。左右に広がるローターの形状は外見から識別しやすく、チューダー コレクターのあいだでもひとつのポイントとして注目されています。

**Q:** バラサブのケースはロレックスと同じなのですか？

**A:** はい。Ref.7928・Ref.7016のチューダー サブマリーナは、ロレックスのオイスターケースを採用しており、裏蓋に「ORIGINAL OYSTER CASE BY ROLEX GENEVA」の刻印があります。王冠リューズもロレックス製のものが使用されており、ケース面ではロレックス サブマリーナと共通の構造です。

**Q:** Ref.7928とRef.7016はいつ製造されたモデルですか？

**A:** Ref.7928は主に1960年代前半から半ばにかけて製造されたモデルで、1963年製・1964年製の個体が確認されています。Ref.7016は1960年代後半から1970年代前半にかけて製造された後継リファレンスです。

**Q:** デカバラと小バラの違いは何ですか？

**A:** 文字盤12時位置の薔薇ロゴのサイズが異なります。Ref.7928の薔薇ロゴは大きく「デカバラ」と呼ばれ、Ref.7016の薔薇ロゴはより小さく「小バラ」と呼ばれます。また、搭載されているキャリバーも異なり、Ref.7928にはバタフライローター搭載のCal.390が使われています。

**Q:** 薔薇サブ（バラサブ）とスノーフレーク（イカサブ）の関係は？

**A:** 薔薇サブはメルセデス針を採用した薔薇ロゴ時代のモデルで、スノーフレーク（イカサブ）は独特の形状のスノーフレーク針と盾ロゴを採用した後継世代のモデルです。薔薇ロゴからスノーフレーク針・盾ロゴへの移行は1960年代後半から1970年代にかけて行われており、チューダー サブマリーナのデザイン転換点を示しています。

## まとめ

チューダー サブマリーナRef.7928とRef.7016は、ロレックスのオイスターケースと王冠リューズを共有しながら、チューダー家の家紋である薔薇ロゴを文字盤に掲げた「バラサブ」の代表的なリファレンスです。Ref.7928のバタフライローター搭載Cal.390と大きな薔薇ロゴ、Ref.7016の小バラロゴと後継ETA製キャリバーという組み合わせは、チューダー サブマリーナの変遷を具体的に示す歴史的な証拠です。

1970年代後半以降に盾ロゴ・スノーフレーク針世代へと移行したことで、バラサブは現在では製造終了から半世紀以上が経過したヴィンテージモデルとして、チューダー愛好家のあいだで根強い支持を集めています。
