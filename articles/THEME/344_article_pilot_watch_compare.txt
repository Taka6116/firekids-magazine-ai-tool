---

パイロットウォッチといえば、航空の現場で実際に使われた機能美の結晶です。本記事では、IWC マーク11、ブライトリング ナビタイマー、ユニバーサル・ジュネーブ ポールルーターという3つの名機を、ムーブメント・機能・設計思想の観点から横断的に比較解説します。

それぞれのブランドが航空時計に込めたアプローチは異なります。耐磁性と堅牢性を徹底したIWC、計算尺という操縦支援ツールを内蔵したブライトリング、航路の特殊性から生まれたマイクロローター技術を持つユニバーサル・ジュネーブ。いずれもヴィンテージ時計の世界で重要な位置を占めるモデルです。

---

## IWC マーク11（Mk.XI）― 軍用耐磁パイロットウォッチの完成形

### 開発の背景

IWC マーク11は、イギリス空軍（Royal Air Force）のパイロットウォッチとして開発・納入されたモデルです。航空機のコックピット内には計器類が発する強い磁気が存在し、通常の時計ではその影響でムーブメントが狂うリスクがありました。マーク11はこの問題を解決するために、耐磁性と防水性を高める二重構造のケースを採用しました。

### Cal.89 ハック付き

マーク11に搭載されるのは、IWCを代表する手巻きムーブメントCal.89のハック機能付きバージョンです。Cal.89は18,000振動/時のロービートキャリバーで、シャフハウゼン一貫製造のマニュファクチュール品として丁寧な仕上げと高い耐久性で知られます。

ハック機能とはリューズを引いた際に秒針を停止させる機構で、軍用時計において時報に合わせた正確な時刻同期を可能にします。Cal.89ハック付きバージョンはマーク11など軍用モデルへの採用に限定されており、通常のCal.89搭載モデルにはハック機能がないことが多い点に注意が必要です。

### マーク11の仕様

| 項目 | 仕様 |
|------|------|
| ムーブメント | 手巻き / Cal.89 ハック付き |
| ケースサイズ | 36mm |
| ケース素材 | ステンレス |
| ケース構造 | 二重構造・スクリューバック |
| 主な納入先 | イギリス空軍（RAF）、オーストラリア空軍（RAAF） |
| 軍リファレンス | 6B/346（RAFモデル） |

### 文字盤と識別ポイント

マーク11の文字盤はパイロットウォッチらしいブラックのマットダイヤルが基本です。イギリス空軍モデルには英国政府所有を示す「ブロードアロー」が文字盤・ムーブメントともに刻印されており、裏蓋には軍リファレンス「6B/346」の刻印が確認できます。

マーク11のバリエーションとして、7のインデックスがフック（鉤）の形状をした「フック7」と呼ばれる希少なダイヤルが知られています。オーストラリア空軍（RAAF）モデルは仕様はRAFモデルと同一ながら、文字盤にブロードアローやTマークが入らない仕様となっています。

## ブライトリング ナビタイマー Ref.806 ― スライドルール内蔵のプロフェッショナルツール

### 開発の背景

ブライトリング ナビタイマーは、AOPA（国際パイロット協会）との連携のもと開発されたパイロットウォッチです。軍用という位置づけのマーク11とは異なり、民間パイロットが航空業務で実際に使うことを前提とした実用ツールとして設計されました。

ナビタイマーを象徴するのが文字盤外周に設けられた回転計算尺（スライドルール）です。速度・距離・時間の換算、燃料消費量の計算など、航空ナビゲーションに必要な計算を文字盤上で行える設計で、機上の計算機として実用されました。

### Cal.Venus178

ナビタイマーRef.806に搭載されたCal.Venus178は、ヴィーナス社製の手巻きクロノグラフキャリバーです。ナビタイマーRef.806のほか、コスモノートRef.809、トップタイムRef.810など、1960年代のブライトリングを代表する複数のモデルに搭載された名機です。

| 項目 | 仕様 |
|------|------|
| ムーブメント | 手巻き / Cal.Venus178 |
| ケースサイズ | 41mm（リューズ別） |
| ラグ幅 | 22mm |
| ケース素材 | ステンレス / GF（金張り） |
| 製造年代 | 1960年代（1965〜1968年製が多い） |

### ナビタイマーRef.806の世代識別

ナビタイマーRef.806には、6時位置のAOPAマークの形状によって複数の世代が存在します。ファーストモデルではAOPAのオリジナルマークが使われ、セカンドモデルからは折り鶴のような航空機2機がランデブーするデザインに変更され、インダイアルも白抜きになります。さらにセカンドモデルの初期型には、AOPAロゴがアップライトになった希少な仕様も確認されています。

### ペンシルハンドと視認性

ナビタイマーRef.806のペンシルハンドは、その名のとおり細く鋭いデザインで、複数のカウンターやスライドルール目盛が並ぶ複雑な文字盤上で時刻を読み取るための視認性を確保しています。直径41mmの大型ケースも、情報量の多い文字盤の視認性に貢献しています。

## ユニバーサル・ジュネーブ ポールルーター ― マイクロローター技術の先駆者

### 開発の背景

ユニバーサル・ジュネーブ ポールルーターは、1954年にSASスカンジナビア航空の北極点経由飛行（ポーラールート）を記念して開発されたモデルです。北極点付近では強い磁場が発生するという特殊な航行環境に対応するため、新型のマイクロローター自動巻きムーブメントが開発されました。

マーク11が軍用機のコックピット内磁気への対策として二重構造ケースを選んだのに対し、ポールルーターは北極圏特有の磁気環境への対応という異なるアプローチから防磁性能を持つ設計を採用しています。

### Cal.138-SS とマイクロローター

ポールルーターに搭載されたCal.138-SSは、商業的に量産に成功した世界初のマイクロローター自動巻きとして時計史に刻まれたキャリバーです。

通常の自動巻き時計はムーブメント全体を覆う大型のローターが回転することで巻き上げを行いますが、Cal.138-SSのマイクロローターはムーブメントの一部に組み込まれた小型のローターです。これによりムーブメント全体を薄くできる点が技術的な革新でした。

| 項目 | 仕様 |
|------|------|
| ムーブメント | 自動巻き / Cal.138-SS（マイクロローター） |
| ケースサイズ | 34〜36mm |
| ケース素材 | ステンレス / 18金 |
| 発表年 | 1954年 |

### ポールルーターの設計とコレクター評価

ポールルーターのケースデザインは、後に多くの名作を生み出すジェラルド・ジェンタが関わったとされています。ロイヤルオークやノーチラス以前の初期ジェンタ仕事として、時計史好きのコレクターからも高い評価を受けています。

ユニバーサル・ジュネーブは現在では新品の製造を行っていないブランドであるため、ポールルーターを含むすべてのモデルがヴィンテージとして流通しています。

### コレクターとしての選択軸

**ミリタリーウォッチとしての歴史を重視するなら：IWC マーク11**

実際に軍に納入され、パイロットが着用した歴史を持つ時計を求めるコレクターにはマーク11が適しています。ブロードアロー刻印やフック7ダイヤルといった識別ポイントを持ち、軍用時計としての証が残っている点が大きな魅力です。Cal.89のハック付きバージョンという希少性も、コレクション価値の一部です。

**実用ツールとしての機能美を求めるなら：ブライトリング ナビタイマー**

スライドルール（回転計算尺）とクロノグラフを組み合わせた、航空ナビゲーションの道具としての機能美を持つのがナビタイマーです。AOPA世代の違いやペンシルハンドのバリエーションなど、ヴィンテージとして研究できる要素が多く、航空史・時計機能史の両面からアプローチできます。

**時計技術史に関心があるなら：UG ポールルーター**

マイクロローター自動巻きの世界初量産機という技術史上の位置づけ、ジェラルド・ジェンタ初期のデザイン仕事、SASポーラールート就航という航空史との接点——ポールルーターはこれら複数の歴史的文脈が交差する時計です。ユニバーサル・ジュネーブが現在は新品を製造していないという希少性も、コレクション対象としての独自性を高めています。

## よくある質問（FAQ）

Q: IWC マーク11のCal.89はどのような特徴を持つキャリバーですか？
A: Cal.89はIWCのシャフハウゼン工場で一貫製造された手巻きムーブメントで、18,000振動/時のロービートキャリバーです。マーク11に搭載されたバージョンにはハック機能（秒針停止機能）が追加されており、軍用時計として求められる正確な時刻同期を可能にしています。ハック機能付きのCal.89はマーク11など軍用モデルへの採用に限定されており、通常のCal.89搭載モデルには備わっていません。

Q: ブライトリング ナビタイマーの回転計算尺はどのように使うのですか？
A: 文字盤外周に設けられた回転スケールは、速度・距離・時間の換算や燃料消費量の計算など、航空ナビゲーションに必要な演算を行うためのツールです。ベゼルを回転させることで目盛りを合わせ、乗算や除算を視覚的に行う計算尺の原理を応用しています。ヴィンテージ個体ではスケールの摩耗や汚れが読み取りに影響することがあります。

Q: ユニバーサル・ジュネーブのポールルーターが「世界初」とされる点は何ですか？
A: ポールルーターに搭載されたCal.138-SSは、商業的に量産に成功した世界初のマイクロローター自動巻きキャリバーとされています。マイクロローターはムーブメントの一部に組み込まれた小型ローターで、通常の大型ローターに比べてムーブメント全体を薄くできる画期的な技術です。

Q: 3モデルのなかで耐磁性能が最も高いのはどれですか？
A: 耐磁性への対策として最も徹底されているのはIWC マーク11です。スクリューバックによる二重構造ケースを採用し、コックピット内の計器類が発する磁気からムーブメントを保護する設計が施されています。ユニバーサル・ジュネーブ ポールルーターも北極圏の磁気環境を意識した設計ですが、対策のアプローチが異なります。

Q: これら3モデルはどのような用途・コレクション軸で選べばよいですか？
A: ミリタリーウォッチとしての歴史的証拠（ブロードアロー・軍リファレンス刻印）を重視するならIWC マーク11、航空ナビゲーションツールとしての機能美と世代識別を楽しむならブライトリング ナビタイマー、技術史（マイクロローター世界初量産）とジェンタ初期デザインに関心があるならユニバーサル・ジュネーブ ポールルーターが適しています。いずれも実際に航空の現場と繋がりを持つ本物のパイロットウォッチです。

## まとめ

IWC マーク11、ブライトリング ナビタイマー、ユニバーサル・ジュネーブ ポールルーター——3モデルはいずれも「航空」という共通の文脈から生まれながら、それぞれがまったく異なる設計思想と歴史的背景を持っています。

マーク11はCal.89ハック付きと二重構造ケースという軍用仕様の完成形として、ナビタイマーはCal.Venus178とスライドルールによる操縦支援ツールとして、ポールルーターはCal.138-SSのマイクロローター技術の先駆として、それぞれ時計史において固有の意味を持っています。

ヴィンテージパイロットウォッチを選ぶ際には、どの設計思想・歴史的背景に惹かれるかを基準にすると、選択がより明確になるでしょう。
