# チューダーの時計とは？ヴィンテージの選び方とおすすめモデル｜FIRE KIDS Magazine

対象ブランド: TUDOR（チューダー）
カテゴリ: チューダー
生成日: 2026.03.13
Written by FIRE KIDS 編集部

---

チューダー（TUDOR）のヴィンテージウォッチを最初の1本として選びたいと考えている方に向けた入門ガイドです。ロレックスのオイスターケースを採用しながら独自のデザインとETA製キャリバーを組み合わせたヴィンテージ・チューダーは、ロレックスにはないカラーリングや針のデザイン、文字盤ロゴの変遷など、独自の収集テーマが豊富なブランドです。

本記事では、ヴィンテージ・チューダーの3大カテゴリ（サブマリーナ / オイスタープリンス系ドレスウォッチ / クロノグラフ）の概要と、各カテゴリにおける入門に向いたリファレンスの特徴を、商品情報をもとに解説します。

---

## チューダーとロレックスの関係はどうなっているのか？

### ロレックスのディフュージョンブランドとして誕生

チューダーは1926年にロレックスの創業者ハンス・ウィルスドルフが設立したブランドです。ロレックスのオイスターケースと王冠リューズを共有しながら、ムーブメントにはETA（Ebauches SA）製キャリバーを採用することでコストを抑えた実用時計として展開されてきた歴史があります。

ヴィンテージのチューダー サブマリーナの裏蓋には「ORIGINAL OYSTER CASE BY ROLEX GENEVA」の刻印が確認でき、ケースがロレックスと共通であることが明示されています。王冠リューズもロレックス製が採用されており、外装面での共通性は高いものがあります。

### ムーブメントの違いが価格差の主因

ロレックスが自社製キャリバー（Cal.1570等）を搭載するのに対し、ヴィンテージのチューダーはETA製キャリバーを採用してきました。ETA製キャリバーは汎用性が高く部品の入手性に優れており、メンテナンス面での実用性は高い評価を受けています。

ケース・リューズはロレックスと共通でありながら、ムーブメントがETA製であることがヴィンテージ市場における価格差の主な要因です。これは最初の1本としてチューダーを選ぶことの大きなメリットの一つです。

## チューダー ヴィンテージにはどんな種類がある？

チューダーのヴィンテージは大きく以下の3カテゴリに分類されます。

| カテゴリ | 代表モデル | 特徴 |
|---|---|---|
| サブマリーナ | 薔薇サブ / イカサブ / ミッドサブ | ダイバーズウォッチ。ロレックスと外装を共有しながらブルーベゼル・スノーフレーク針など独自性が強い |
| オイスタープリンス系（ドレスウォッチ） | オイスターローヤル / プリンスオイスターデイト | 薔薇ロゴ・盾ロゴのドレスウォッチ。文字盤カラーの多様性が魅力 |
| クロノグラフ | Monte-Carlo / クロノタイム | バルジュー製手巻きクロノ（モンテカルロ）とETA Cal.7750搭載のクロノタイム系 |

## カテゴリ1：チューダー サブマリーナ

### サブマリーナのロゴ・世代別マップ

チューダー サブマリーナはロゴと針の形状で3世代に大きく区分されます。

| 世代 | ロゴ | 針 | 代表Ref. | 時代 |
|---|---|---|---|---|
| 薔薇サブ（バラサブ） | 薔薇ロゴ | メルセデス針 | Ref.7928 / Ref.7016 | 〜1970年代前半 |
| イカサブ（スノーフレーク） | 盾ロゴ | スノーフレーク針 | Ref.7021/0 / Ref.9411/0 / Ref.79090 | 1960年代後半〜1990年代 |
| ミッドサブ | 盾ロゴ | メルセデス針 | Ref.75090 / Ref.75190 | 1988年頃〜 |

### 薔薇サブ（バラサブ）

薔薇サブは文字盤12時位置に薔薇ロゴを持つ、1970年代前半以前のチューダー サブマリーナです。薔薇のサイズで「デカバラ（大きな薔薇）」と「小バラ（小さな薔薇）」に区分されます。

**Ref.7928（デカバラ サブマリーナ）**

| 項目 | 仕様 |
|---|---|
| 製造年代 | 1960年代（1963〜1964年製の個体が確認されている） |
| ムーブメント | 自動巻き / Cal.390（バタフライローター搭載） |
| ケースサイズ | 40mm |
| ケース素材 | ステンレス |
| ラグ幅 | 20mm |
| ベゼル | ブラック（ロング5ベゼル） |
| 裏蓋刻印 | ORIGINAL OYSTER CASE BY ROLEX GENEVA |
| ブレスレット | ロレックス製3連ステンレスブレス / リベットブレス |

Cal.390は、バタフライローターと呼ばれる蝶の羽状の形状を持つ特徴的な自動巻きキャリバーです。ロレックスのCal.1570（自社製）と比較すると、このバタフライローターのビジュアルがコレクターの注目ポイントになっています。

薔薇サブは製造から60年以上が経過したモデルであるため、入門の1本として選ぶには文字盤のオリジナル性（リダンの有無）、夜光の経年変化の均一性など、確認事項が多くあります。入門として選ぶ場合は、状態の説明が明確な信頼できる専門店での購入が重要です。

**Ref.7016（小バラ サブマリーナ）**

Ref.7016は1960年代後半から1970年代前半にかけて製造された、薔薇ロゴ世代の最終リファレンスです。40mmオイスターケース、ブラックベゼルという基本仕様はRef.7928と共通ながら、薔薇ロゴがより小さく控えめなデザインとなっています。

薔薇サブの入門として選ぶなら、Ref.7928よりも製造年代が新しく状態の良い個体が比較的見つかりやすいRef.7016は、コストと状態のバランスを取りやすい選択肢です。

---

### イカサブ（スノーフレーク サブマリーナ）

イカサブはスノーフレーク針（日本では「イカ針」の愛称）と盾ロゴを採用したチューダー サブマリーナの世代で、1960年代後半から1990年代にかけて製造されました。ロレックスには存在しないブルーベゼルとブルーダイヤルの組み合わせがイカサブ最大の個性です。

**入門に向いたRef.9411/0（1970年代）**

| 項目 | 仕様 |
|---|---|
| 製造年代 | 1976〜1980年製の個体が確認されている |
| ムーブメント | 自動巻き / Cal.2784（ETA系） |
| ケースサイズ | 40mm |
| ケース素材 | ステンレス |
| ラグ幅 | 20mm |
| ベゼル | ブルー / ブラック |
| ブレスレット | 純正巻きブレス9315/FF380B |

Ref.9411/0はイカサブの中で最も人気が高いリファレンスです。ブルーベゼル×ブルーダイヤルの組み合わせはブラックダイヤルより希少ですが、ブラックダイヤル仕様はイカサブ入門として入手しやすい選択肢となっています。

Cal.2784はETA系の信頼性の高い自動巻きキャリバーで、オーバーホール部品の入手性が高い点は実用面でのメリットです。

**後期型：Ref.79090（1980年代後半〜1990年代）**

| 項目 | 仕様 |
|---|---|
| 製造年代 | 1992〜1993年製の個体が確認されている |
| ムーブメント | 自動巻き / Cal.2824-2 |
| ケースサイズ | 40mm |
| ベゼル | ブラック |
| ブレスレット | 純正巻きブレス9315/FF380B |

Ref.79090は1990年代の後期イカサブです。製造年代が新しいため状態の良い個体を見つけやすい点で、入門の1本として実用性が高いリファレンスです。フチなし夜光と巻きブレスの組み合わせは、1970年代モデルに通じるヴィンテージ感を持っています。

**Ref.75090（1988年頃〜）**

| 項目 | 仕様 |
|---|---|
| 製造年代 | 1988〜1993年製の個体が確認されている |
| ムーブメント | 自動巻き / Cal.2824-2 |

「ミッドサブ」と呼ばれる36mmケースのサブマリーナは、40mmが主流のサブマリーナカテゴリにおいてロレックスには存在しないサイズ展開です。腕の細い方や、コンパクトなサイズを好む方にとっては独自のポジションを持つ選択肢です。搭載するCal.2824-2はETA製自動巻きとして信頼性が高く、実用的な入門モデルです。

## カテゴリ2：オイスタープリンス系（ドレスウォッチ）

オイスターローヤル（Oyster Royal）とプリンスオイスターデイト（Prince OysterDate）は、1950〜1970年代のチューダーを代表するドレスウォッチです。サブマリーナとは異なる薔薇ロゴ・盾ロゴの変遷と、ロレックスにはない多彩な文字盤カラーが収集テーマとして魅力的なカテゴリです。

### オイスターローヤル

| リファレンス | 年代 | ムーブメント | ケースサイズ |
|---|---|---|---|
| Ref.7903 | 1950年代 | 手巻き | 約31mm |
| Ref.7904 | 1955年頃 | 手巻き / Cal.1182 | 34mm |

手巻きのベーシックモデルです。31〜34mmのコンパクトなケースサイズは現代でもユニセックスに着用できます。

### プリンスオイスターデイト

| リファレンス | 年代 | ムーブメント | ケースサイズ | 特徴 |
|---|---|---|---|---|
| Ref.7966 | 1964年頃 | 自動巻き / Cal.2484 | 34mm | デカバラ、クサビインデックス |
| Ref.7996/0 | 1968年頃 | 自動巻き | 34mm | 小薔薇、シルバーダイヤル |
| Ref.7106/0 | 1969年製 | 自動巻き / Cal.2772 | 34mm | ブルーダイヤル |
| Ref.9052 | 1980年製 | 自動巻き / Cal.2784 | 34mm | 盾ロゴ |
| Ref.74000 | 1997年製 | 自動巻き / Cal.2824-2 | 34mm | ダークブルーダイヤル |

自動巻き＋デイト機能を搭載した上位モデルです。文字盤カラーが多彩で、ロレックスにはないブルーやシルバーなどのバリエーションが特徴です。

1964年製Ref.7966はデカバラの立体的な薔薇ロゴが最大の魅力ですが、歴史的に人気が高くオリジナルコンディションの個体は希少性があります。入門として選びやすいのは、製造年代が新しく状態の良い個体が見つかりやすいRef.9052（Cal.2784搭載・盾ロゴ）やRef.74000（Cal.2824-2搭載）です。

## カテゴリ3：クロノグラフ

### Monte-Carlo（モンテカルロ）

チューダーのヴィンテージ・クロノグラフの中でも特に個性的な文字盤デザインで知られるシリーズです。ロレックスのオイスターケースを採用しながら、ロレックスにはないカラーサブダイヤルを組み合わせた独自のデザインが特徴です。

**Ref.7159/0（確認済み仕様）**

| 項目 | 仕様 |
|---|---|
| 製造年代 | 1970年代前半（1972年製の個体が確認されている） |
| ムーブメント | 手巻き / バルジュー社 Cal.234 |
| プッシャー | ねじ込み式プッシャー |
| ブレスレット | ロレックス純正7836（FF382） |

Cal.234はバルジュー（Valjoux）社が供給した手巻きクロノグラフ用キャリバーです。ねじ込み式プッシャーの状態がコンディション評価の重要な指標となります。

### クロノタイム系（Cal.7750搭載）

Monte-Carloシリーズの後継として展開されたクロノタイム系は、ETA Cal.7750自動巻きクロノグラフを搭載したシリーズです。

| リファレンス | 年代 | 特徴 |
|---|---|---|
| Ref.79160 | 1989年〜 | ビッグブロック、前期は枠ありカレンダー |
| Ref.79180 | 1990年代前半〜 | パンダ・逆パンダ・ブラック各種の文字盤 |
| Ref.79260 | 1996年製確認 | 後期型、王冠リューズ |
| Ref.79280 | 1995年〜 | 純正オイスターブレス、3連 |

クロノグラフカテゴリの入門として選びやすいのはクロノタイム系です。Cal.7750は汎用性の高い自動巻きクロノグラフ・キャリバーとしてオーバーホール対応店が多く、実用性の面での安心感があります。文字盤はパンダ（白地に黒サブダイヤル）・逆パンダ（黒地に白サブダイヤル）・ブラックなど複数のバリエーションが展開されました。

## チューダーとロレックスは何が違うのか？

### 共通する要素

- **オイスターケース**：ねじ込み式裏蓋と王冠リューズによる防水構造（裏蓋に「ORIGINAL OYSTER CASE BY ROLEX GENEVA」刻印）
- **王冠リューズ**：ロレックス製の王冠リューズを使用した個体が多い
- **ブレスレット**：ロレックス純正ブレスが装着された個体が確認されている

### チューダーならではの独自性

| 項目 | チューダーの独自性 |
|---|---|
| ムーブメント | ETA製キャリバー（Cal.390 / 2484 / 2784 / 2824-2 / 7750等） |
| 文字盤ロゴ | 薔薇ロゴ（デカバラ・小バラ）→ 盾ロゴ（ホームプレート）への変遷 |
| 針のデザイン | スノーフレーク針（イカ針）はロレックスに存在しない |
| ベゼルカラー | ブルーベゼルはチューダー サブマリーナ独自 |
| ケースサイズ | 36mmミッドサブはロレックス サブマリーナには存在しない |
| 文字盤カラー | ドレスウォッチのブルー・ピンク等の多彩なカラーバリエーション |

## 購入時のチェックポイント

### ロゴの確認

チューダーのヴィンテージはロゴの時代変遷が重要な情報です。文字盤のロゴ（薔薇か盾か）と、リューズや裏蓋のロゴを照合することで製造年代の確認ができます。

- 薔薇ロゴ時代（〜1970年代前半）：デカバラ → 小バラの変遷
- 盾ロゴ時代（1970年代後半〜）：盾ロゴが標準

なお、同一Ref.でも複数のロゴバリエーションが存在するケースがあります。より詳細な年代特定にはRef.番号・キャリバー番号・シリアル番号の照合が必要です。

### ケースコンディションの確認

オイスターケースはロレックスと共通の堅牢な構造ですが、過去の研磨歴によってエッジが痩せている個体も存在します。ケースの角（エッジ）の立ち具合、ラグ部分の状態は購入前に確認すべきポイントです。

### 文字盤のオリジナル性（リダンの有無）

ヴィンテージ・チューダーには、文字盤がリダン（再塗装）された個体が一定数存在します。薔薇サブやイカサブはロゴや針の形状が文字盤のアイデンティティそのものであるため、リダンの有無はコレクション価値を左右します。

オリジナル文字盤かどうかの確認方法：
- 夜光インデックスと針の経年変化が均一かどうか
- 薔薇ロゴや盾ロゴの印刷精度がオリジナルらしいか
- 信頼できる専門店のリダン有無に関する記載を確認する

### ムーブメントのオーバーホール歴

入門の1本としては、オーバーホール済みの個体を選ぶことを推奨します。特に製造から40〜60年以上経過したRef.7928・Ref.7021/0・Ref.9411/0等の個体は、オーバーホール未施工の場合、精度や動作に問題が生じている可能性があります。

ETA製キャリバー（Cal.2784・Cal.2824-2等）は汎用性が高く、オーバーホールの対応店が多いという実用面での利点があります。

## よくある質問

**Q:** ヴィンテージ・チューダーのムーブメントはロレックスより品質が劣りますか？

**A:** ケース・リューズはロレックスと共通部品が使われており、外装の品質はロレックスと同水準です。ムーブメントはETA製キャリバーで、ロレックスの自社製キャリバーとは異なりますが、ETAは信頼性の高いスイス製汎用ムーブメントのサプライヤーとして広く知られており、汎用性と部品入手性の高さは実用面でのメリットです。

**Q:** イカサブのブルーベゼルとブラックベゼル、どちらが入門向きですか？

**A:** 一般的にブルーベゼル×ブルーダイヤルの組み合わせが希少性が高く、ブラックダイヤル仕様が入手しやすい傾向にあります。入門としてはブラックダイヤルのRef.9411/0や後期型のRef.79090が選びやすい選択肢です。ブルーベゼル・ブルーダイヤルはその希少性もイカサブの魅力の一つとして知られています。

**Q:** 薔薇サブとイカサブ、どちらが入門として向いていますか？

**A:** 入門としては、製造年代が新しく状態の良い個体が見つかりやすいイカサブ（特にRef.9411/0ブラックダイヤルやRef.79090）が選びやすい傾向にあります。薔薇サブ（特にRef.7928）は製造から60年以上が経過しており、オリジナルコンディションの確認が重要になります。薔薇ロゴそのものへの強い関心がある場合は、専門店での詳細な状態確認を前提に選択することが大切です。

**Q:** ミッドサブ（36mm）はどのような方に向いていますか？

**A:** 腕周りの細い方や、ダイバーズウォッチとしてのデザインを日常的に着用したいがより小さなサイズを希望する方に向いています。ロレックスのサブマリーナには存在しない36mmというサイズは、チューダーのヴィンテージ・サブマリーナが持つ独自のラインナップです。Cal.2824-2搭載のRef.75090は、実用性と入手しやすさのバランスが取れた入門向きの選択肢です。

**Q:** チューダーのヴィンテージを購入する際、最初に確認すべきことは何ですか？

**A:** 文字盤のオリジナル性（リダンの有無）、ムーブメントのオーバーホール歴、ケースコンディション（研磨歴によるエッジの状態）の3点が特に重要です。また、リューズや裏蓋の刻印・状態、ベゼルインサートの状態（退色・交換の有無）も確認の対象です。状態の説明が詳細で透明性の高い専門店を選ぶことが、最初の1本選びでは特に大切です。

## まとめ

チューダーのヴィンテージは、ロレックスのオイスターケースという堅牢な土台を持ちながら、スノーフレーク針・ブルーベゼル・薔薇ロゴの変遷・多彩な文字盤カラーなど、ロレックスには存在しない豊かな個性を持つコレクションです。ETA製キャリバーの採用はムーブメントのメンテナンス面での実用性に結びつく要素でもあり、最初の1本としての選択肢として考える意義があります。

サブマリーナ・ドレスウォッチ・クロノグラフという3大カテゴリの中から、自分の関心（ダイバーズウォッチ / ドレスウォッチ / クロノグラフ）と、求める時代感（薔薇サブの1960年代 / イカサブの1970〜90年代 / ドレス系の多彩な文字盤）に合ったカテゴリを選ぶことが、チューダーヴィンテージ入門の出発点となります。
