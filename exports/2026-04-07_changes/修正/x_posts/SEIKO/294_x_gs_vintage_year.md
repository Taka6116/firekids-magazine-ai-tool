グランドセイコー ヴィンテージの年代を見分ける方法——ケース・ブレスレット・リファレンス番号の読み方

手元にあるヴィンテージ グランドセイコーがいつ頃製造されたものか。ケース形状・素材・リファレンス番号・シリアル番号を組み合わせることで、製造年代を精度高く絞り込むことができます。

ケース形状で見る4世代の変遷

ヴィンテージ グランドセイコーのケースは大きく4つの世代に分けられます。

第1世代：ラウンドケース（1960〜1963年）
初代グランドセイコー Ref.J14070。Cal.3180（25石・手巻き・18,000振動/時）を搭載し、ケースサイズ34〜35mm。素材は14KGF（14金張り）、裏蓋はスナップバック方式です。丸みを帯びた古典的なシルエットが特徴で、14KGFのケースを持つグランドセイコーはすべてこの世代と判定できます。

第2世代：スクリューバック・ラウンドケース（1964〜1966年）
57GS（Ref.5722-9991 / Ref.5722-9011、Cal.5722）とGS初の自動巻きモデル62GS（Ref.6245-9000 / Ref.6246-9001、Cal.6245/6246）。ケース素材がステンレスまたはキャップゴールドに変わり、裏蓋がスクリューバックに改められました。1stモデルとの最も明確な違いは裏蓋の構造です。

第3世代：セイコースタイルケース（1967〜1969年）
44GS（Ref.4420-9000、Cal.4420）・45GS（Ref.4522-8000 / Ref.4520-8000、Cal.4522/4520）・61GS（Ref.6145-8000 / Ref.6146-8000、Cal.6145/6146）。平面と曲面の稜線を鋭く仕上げた「セイコースタイル」が確立された時代です。ラグとケース胴の境目に明確な稜線が走っているのが判断基準になります。

第4世代：バレルケース（1971〜1972年）
56GS（Ref.5645-7010 / Ref.5646-7010 / Ref.5641-7000 / Ref.5646-5010）。ケース全体が樽型のシルエットで、セイコースタイルの鋭い稜線とは対照的な有機的な曲線が特徴です。

ケース素材と年代の関係

・14KGF（14金張り）：1stモデルのみ（1960〜1963年）
・キャップゴールド：57GS Ref.5722-9011（1964〜1967年）、44GS Ref.4420-9990
・ステンレス（SS）：1964年以降すべての世代
・18K（18金無垢）：45GS Ref.4522-8010（1967〜1968年）の一部特別仕様

14KGFの刻印があれば1stモデルと即座に判定できます。ステンレスは幅広い年代に使われているため、素材だけでは絞り込めません。ケース形状・リファレンス番号との組み合わせが必要です。

リファレンス番号の読み方

グランドセイコーのリファレンス番号は「キャリバー番号-型式番号」という構成です。先頭4桁がキャリバー番号に直接対応しており、ここからモデル名と製造年代を逆引きできます。

・5722-XXXX → Cal.5722（57GS・手巻き・18,000振動/時・1964年〜）
・6245-XXXX / 6246-XXXX → Cal.6245/6246（62GS・自動巻き・ロービート・1967年〜）
・4420-XXXX → Cal.4420（44GS・手巻き・ハイビート・1967年〜）
・4522-XXXX / 4520-XXXX → Cal.4522/4520（45GS・手巻き・ハイビート±2秒/日・1968年〜）
・6145-XXXX / 6146-XXXX → Cal.6145/6146（61GS・自動巻き・ハイビート・1968年〜）
・5641-XXXX / 5645-XXXX / 5646-XXXX → Cal.5641/5645/5646（56GS・自動巻き・ハイビート・1970年〜）

リファレンス番号の後半4桁は、ケース素材・文字盤カラー・デイト有無などのバリエーションを示します。たとえばRef.5646-5010のようにバレルケース版に固有の番号が割り当てられています。

文字盤の仕上げと年代

文字盤の仕上げも年代判定の手がかりになります。

・彫り文字盤（「Grand Seiko」が凹型彫刻）：1stモデル前期（1960年頃）
・SDダイヤル・アップライトロゴ：1stモデル中期（1961〜1962年頃）
・ライオンメダリオン付き：1stモデル後期（1963年頃）
・カレンダー窓あり・シルバー系：57GS・62GS（1964〜1967年）
・バー・クサビインデックス（セイコースタイル）：44GS・45GS・61GS（1967〜1969年）
・クロスライン仕上げ（縦横の筋目）：61GS・56GS（1968〜1972年）
・絹目・サンバースト・グラデーション：56GS（1971〜1972年）

56GS時代は文字盤バリエーションが最も豊富で、Ref.5646-5010のバレルケース版ではグラデーションダイヤルも確認されています。

年代判定のステップ

ヴィンテージ グランドセイコーを前にしたときの確認手順は以下の通りです。

ステップ1：ケース素材を確認する
14KGF → 1stモデル（1960〜1963年）確定。ステンレスならステップ2へ。

ステップ2：ケース形状を確認する
スナップバック・ラウンド → 1stモデル
スクリューバック・ラウンド → 57GS・62GS（1964〜1967年）
稜線鋭いセイコースタイル → 44GS・45GS・61GS（1967〜1969年）
バレル型・丸みのある輪郭 → 56GS（1971〜1972年）

ステップ3：リファレンス番号の先頭4桁でキャリバーを特定する
5722 / 6245 / 6246 / 4420 / 4522 / 4520 / 6145 / 6146 / 5641 / 5645 / 5646 のいずれかからモデルと登場年を確定させます。

ステップ4：シリアル番号で製造年を絞り込む
裏蓋または中留内側のシリアル番号の先頭数字を確認します。先頭「1」は1971年、「2」は1972年の製造が目安です。ただしキャリバー登場年との整合性を必ず確認してください。

━━━━━━━━━━━━━━

ケース・リファレンス・文字盤の3つの要素を組み合わせることで、ヴィンテージ グランドセイコーの製造年代はかなり正確に絞り込めます。特に「リファレンス先頭4桁=キャリバー番号」という法則を覚えておくと、現物を前にした判断が格段に速くなります。
