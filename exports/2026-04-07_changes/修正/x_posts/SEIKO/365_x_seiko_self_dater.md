# セイコー セルフデーター解説

セイコーの「セルフデーター（Self-Dater）」という名称をご存じでしょうか。日付を自動表示する機構を持つ時計に与えられたセイコー独自の呼称で、1950年代初頭から採用されたシリーズです。グランドセイコーから一般グレードまで幅広く展開されたこのシリーズについて、確認できたデータをもとに解説します。

━━━━━━━━━━━━━━

「セルフデーター」とは何か

セルフデーターとは、ムーブメントの動力を利用して日付リングを毎日自動的に送る機構を持つ腕時計の呼称です。外部操作なしに日付が切り替わる点が「Self（自ら）」という命名の由来となっています。

現代では当然の機能も、1950年代の国産腕時計市場では先進的な仕様でした。モデルによってはケース側面のプッシュボタンでデイトを早送りできる機構も備わっており、月末の手動修正を可能にしています。

━━━━━━━━━━━━━━

スーパーセルフデーター（1954年）

確認できる最初期のセルフデーターは1954年製のスーパーセルフデーター（SUPER SELF DATOR）です。

・リファレンス：Ref.13018
・ムーブメント：手巻き / Cal.970
・ケース素材：ステンレス
・ケースサイズ：31.5mm（リューズ別）
・ラグ幅：16mm

Cal.970を搭載したこのモデルは、コンパクトなケースに先進的なデイト機構を搭載した1本です。4時位置のプッシュボタンによるデイト早送り機構も備えており、当時としては実用的な仕様でした。

━━━━━━━━━━━━━━

クラウン・クロノス・マチックへの展開

1960年代に入るとセルフデーターの機構は様々なグレードに広がります。

クラウン セルフデーター（1960年頃）
・リファレンス：Ref.J14036
・ムーブメント：手巻き
・クラウンシリーズのデイト付きバリエーション

クロノス セルフデーター（1962〜1963年頃）
・リファレンス：Ref.J15016E
・ケース素材：EGP（電気金メッキ）・ステンレス
・ムーブメント：手巻き

マチックセルフデーター（1963年頃）
・リファレンス：Ref.J13060
・ムーブメント：自動巻き・39石
・自動巻き世代のセルフデーター

マチックセルフデーターは自動巻きとデイト機構を組み合わせた一段上のグレードで、39石という高石数のムーブメントを搭載しています。

グランドセイコー セルフデーター

グランドセイコーにもセルフデーターの名を持つバリエーションが存在します。

GS 2ndモデル セルフデーター（1963〜1965年）
・リファレンス：Ref.43999
・ムーブメント：手巻き / Cal.430（35石）
・ケースサイズ：36〜36.5mm
・ラグ幅：19mm
・スクリューバック

GS 57GS セルフデーター（1964〜1967年）
・リファレンス：Ref.5722-9990
・ムーブメント：手巻き / Cal.5722B（35石、18,000振動/時）
・クロノメーター規格対応

Cal.430からCal.5722Bへの移行は精度向上を示しており、「クロノメーター」表記の入った文字盤バリエーションも存在します。裏蓋の獅子メダリオン（ライオンメダリオン）がGSセルフデーターの象徴的なディテールです。

「SDダイヤル」と呼ばれる6時位置のデイト窓に「SD」の文字が入った文字盤も、Ref.43999の初期型などに見られる特徴的なバリエーションです。

まとめ

セイコー セルフデーターは1950年代から1960年代にかけて、スーパーセルフデーター（Cal.970）に始まり、クラウン・クロノスの中堅ライン、自動巻きのマチックセルフデーター、そしてグランドセイコー 2ndモデルと、各時代・各グレードを網羅したシリーズです。

グレードによってムーブメントのスペックや仕上げが大きく異なるため、リファレンス番号とキャリバーを軸に探求すると、当時のセイコーの技術開発の歩みが見えてきます。
