# チューダー ヴィンテージ入門｜最初の1本の選び方

チューダー（TUDOR）のヴィンテージウォッチを初めて購入しようとしている方へ。ロレックスのオイスターケースを使いながら、ロレックスにはない独自のデザインとETA製キャリバーを組み合わせたヴィンテージ・チューダーは、独自の収集テーマが豊富なブランドです。

本稿ではサブマリーナ / ドレスウォッチ / クロノグラフという3大カテゴリの概要と、各カテゴリにおける入門向けリファレンスを整理します。

チューダーとロレックスの関係

チューダーは1926年にロレックス創業者ハンス・ウィルスドルフが設立したブランドです。ヴィンテージのチューダー サブマリーナの裏蓋には「ORIGINAL OYSTER CASE BY ROLEX GENEVA」の刻印が確認でき、ケースはロレックスと共通です。王冠リューズもロレックス製が採用されていました。

ロレックスが自社製キャリバー（Cal.1570等）を搭載するのに対し、チューダーはETA製キャリバーを採用してきました。ケースはロレックスと共通でありながらムーブメントがETA製であることが価格差の主な要因です。これが最初の1本としてチューダーを選ぶ大きなメリットの一つになっています。

ETA製キャリバーは汎用性が高く部品の入手性に優れており、オーバーホール対応店が多いという実用面での安心感があります。

カテゴリ1：チューダー サブマリーナ

サブマリーナはロゴと針の形状で3世代に大きく区分されます。

・薔薇サブ（バラサブ）：薔薇ロゴ＋メルセデス針。代表Ref.7928（デカバラ）/ Ref.7016（小バラ）。1970年代前半まで。
・イカサブ（スノーフレーク）：盾ロゴ＋スノーフレーク針。代表Ref.9411/0 / Ref.79090。1960年代後半〜1990年代。
・ミッドサブ：盾ロゴ＋メルセデス針。Ref.75090（36mm）。1988年頃〜。

薔薇サブのRef.7928はCal.390（バタフライローター搭載の自動巻き）を搭載し、1960年代の40mmオイスターケースにデカバラが輝く象徴的なモデルです。製造から60年以上が経過しており、オリジナル文字盤の確認が特に重要になります。

イカサブの入門として選びやすいのはRef.9411/0のブラックダイヤル仕様と後期型のRef.79090です。Ref.9411/0はCal.2784搭載、Ref.79090はCal.2824-2搭載と、どちらもETA系の実用的なキャリバーです。

ブルーベゼル×ブルーダイヤルの組み合わせはロレックス サブマリーナには存在しないチューダー独自のカラーリングで、ブラックダイヤル仕様と比べると希少性が高い傾向にあります。

ミッドサブのRef.75090は36mmケースとCal.2824-2の組み合わせで、コンパクトなサイズを好む方やロレックスには存在しない独自サイズを求める方に向いた選択肢です。

カテゴリ2：オイスタープリンス系（ドレスウォッチ）

オイスターローヤル（Oyster Royal）とプリンスオイスターデイト（Prince OysterDate）は、1950〜1970年代のチューダーを代表するドレスウォッチです。

プリンスオイスターデイトの代表的なリファレンス：

・Ref.7966（1964年頃）：デカバラ＋Cal.2484搭載・34mm。薔薇ロゴの人気モデル。
・Ref.7106/0（1969年製）：ブルーダイヤル＋Cal.2772搭載。ロレックスにないカラー。
・Ref.9052（1980年製）：盾ロゴ＋Cal.2784搭載。実用的な盾ロゴ時代の1本。
・Ref.74000（1997年製）：ダークブルーダイヤル＋Cal.2824-2搭載。入門向きの後期型。

ドレスウォッチとしての入門には、製造年代が新しく状態の良い個体が見つかりやすいRef.9052やRef.74000が選びやすい選択肢です。薔薇ロゴへの強い関心がある場合は1964年製Ref.7966のデカバラが代表的ですが、オリジナルコンディションの確認が重要です。

カテゴリ3：クロノグラフ

チューダーのヴィンテージ・クロノグラフには「モンテカルロ（Monte-Carlo）」と「クロノタイム」の2系統があります。

モンテカルロ（Ref.7159/0）はバルジュー社Cal.234搭載の手巻きクロノグラフで、1972年製の個体が確認されています。ねじ込み式プッシャーとロレックス純正ブレス（7836/FF382）を備えたヴィンテージ・クロノグラフです。ロレックスにはないカラーサブダイヤルが最大の特徴です。

クロノタイム系はETA Cal.7750自動巻きクロノグラフを搭載したシリーズです。

・Ref.79160（1989年〜）：ビッグブロック、Cal.7750搭載
・Ref.79180（1990年代前半〜）：パンダ・逆パンダ・ブラック各種
・Ref.79260（1996年製確認）：後期型、王冠リューズ

クロノグラフカテゴリの入門としてはクロノタイム系が実用的です。Cal.7750は汎用クロノグラフ・キャリバーとしてオーバーホール対応店が多く、文字盤のバリエーション（パンダ / 逆パンダ / ブラック）を収集テーマにすることもできます。

購入前に確認すべきポイント

ヴィンテージ・チューダーを購入する際に特に重要な確認事項を整理します。

・文字盤のオリジナル性（リダン有無）：薔薇サブ・イカサブはロゴと針が文字盤のアイデンティティそのもの。夜光の経年変化の均一性が判断材料の一つ。
・ムーブメントのオーバーホール歴：製造から40〜60年以上経過したモデルはオーバーホール済み個体を推奨。
・ケースコンディション：研磨歴によるエッジの状態、ラグ部分の確認。
・ベゼルインサート：交換の有無と退色の状態。

チューダーのロゴ変遷（薔薇ロゴ→盾ロゴ）は製造年代を判断する重要な手がかりです。ただし同一Ref.で複数のバリエーションが存在するケースもあるため、Ref.番号・キャリバー番号・シリアル番号を総合的に確認することが正確な年代判定につながります。

カテゴリ別 入門Ref.まとめ

・薔薇サブ入門：Ref.7016（小バラ）。薔薇サブとして状態良個体が比較的見つかりやすい。
・イカサブ入門：Ref.9411/0（ブラックダイヤル）/ Ref.79090。Cal.2784・2824-2の実用性。
・ミッドサブ：Ref.75090。36mmという独自サイズ。Cal.2824-2搭載。
・ドレスウォッチ入門：Ref.9052 / Ref.74000。盾ロゴ時代の実用的なプリンスオイスターデイト。
・クロノグラフ入門：Ref.79160 / Ref.79180（クロノタイム）。Cal.7750のメンテナンス対応店が多い。
